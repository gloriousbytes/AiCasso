Guide d'utilisation de la fonction Image en Vidéo

La fonction Image en Vidéo d'AiCasso vous permet de transformer une seule image en une vidéo dynamique. Décrivez simplement ce que vous voulez voir dans votre vidéo, et AiCasso le réalisera pour vous !

Prompt 1 :

Paramètres Optionnels

Si vous souhaitez être plus précis avec votre vidéo, voici quelques réglages que vous pouvez ajuster :

  • Prompt Négatif : Décrivez ce que vous *ne* voulez pas dans la vidéo. Si laissé vide, AiCasso se concentrera uniquement sur le prompt positif.
  • Nombre de Frames : Choisissez combien de frames (images individuelles) votre vidéo aura. Vous pouvez sélectionner jusqu'à 25 frames, avec 16 par défaut.
  • Étapes de Dénonciation : Cela contrôle la fluidité de votre vidéo. Plus d'étapes signifient une vidéo plus propre. Vous pouvez définir jusqu'à 50 étapes, avec 20 par défaut.
  • Échelle de Guidage : Cela aide la vidéo à coller de près à votre description. La plage est de 1.0 à 8.0, et vous pouvez ajuster les valeurs de début et de fin pour guider l'évolution de la vidéo.
  • Mouvement : Ajustez la quantité de mouvement dans votre vidéo. Une valeur plus élevée signifie plus d'action. La plage est de 0 à 127, avec 60 par défaut.
  • Force d'Augmentation du Bruit : Ajoutez du bruit à votre vidéo pour lui donner un aspect plus texturé. La plage est de 0 à 1, avec 0 par défaut.
  • Images par Seconde (FPS) : Définissez la vitesse de lecture de votre vidéo. Le défaut est de 16 FPS, mais vous pouvez aller jusqu'à 16 pour une lecture plus rapide.
  • Méthode d'Interpolation : Choisissez comment les frames sont mélangées ensemble. Le défaut est "lerp" (interpolation linéaire), mais vous pouvez également utiliser "slerp" (interpolation linéaire sphérique) pour des effets différents.
  • Saut de Clip : Sautez certaines couches pour un style de vidéo différent. Vous pouvez sauter jusqu'à 2 couches, ou laisser null pour le comportement par défaut.

Rappelez-vous, AiCasso est assez intelligent ! Vous n'avez pas besoin d'être parfait dans chaque détail ou d'utiliser une grammaire correcte. Décrivez simplement ce que vous voulez, et AiCasso fera le reste. Amusez-vous à créer vos vidéos !
