Conseils pour la Fonction de Repeinture

Utiliser la fonction de repeinture dans AiCasso vous permet de modifier des zones spécifiques d'une image en les remplissant ou en les remplaçant. Que vous souhaitiez améliorer une image existante ou modifier complètement certaines sections, AiCasso rend le processus simple. Sélectionnez simplement les zones que vous souhaitez changer, fournissez une invite, et laissez AiCasso faire sa magie.

Options :

Intensité :

L'option d'intensité contrôle combien de l'image originale est modifiée par l'invite. Elle varie de 0 à 1. Une valeur de 0 signifie que l'image originale est à peine altérée, tandis qu'une valeur de 1 signifie que l'image originale est presque complètement remplacée par les nouveaux détails que vous décrivez. Pensez-y comme à la 'force' de vos changements.

Exemples :

Invite 1 : "Une image hyperréaliste d'un chat dans le parc."

Résultat : AiCasso générera un chat hyperréaliste dans le parc, remplissant les zones peintes de manière transparente avec des textures et un éclairage réalistes.

Invite 2 : "Un chat dans le parc avec un style hyperréaliste et une intensité de 0,7."

Résultat : Cela créera un chat hyperréaliste dans le parc, avec une intensité de modification réglée à 0,7, intégrant les nouveaux éléments de manière fluide dans l'image originale.
