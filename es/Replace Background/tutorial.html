Guía para Usar la Función de Reemplazo de Fondo

La función de Reemplazo de Fondo en AiCasso es perfecta para cuando deseas cambiar el fondo de tu imagen. Ya sea que solo quieras eliminar el fondo o reemplazarlo con algo más, AiCasso lo hace muy fácil. No necesitas preocuparte por cosas técnicas: solo describe lo que quieres y AiCasso hará el resto.

Parámetros Opcionales

Si deseas agregar un toque personal o ajustar los resultados, aquí hay algunas opciones con las que puedes experimentar:

  • Solo Eliminar: Si solo deseas eliminar el fondo y no reemplazarlo con nada, esta es tu opción ideal. Es como presionar un botón de borrón y cuenta nueva.
  • Matizado Alfa: Esta es una forma elegante de decir "mezclar los bordes". Cuando lo activas, ayuda a que los bordes de tu sujeto se vean más suaves y menos recortados. Por defecto, está desactivado.
  • Umbral de Primer Plano para Matizado Alfa: Esto controla cuánto del sujeto principal (como una persona) permanece visible al usar el matizado alfa. El valor predeterminado es 240, y puedes ajustarlo para hacer los bordes más suaves o más definidos.
  • Umbral de Fondo para Matizado Alfa: Esto hace lo mismo que el umbral de primer plano, pero para el fondo. El valor predeterminado es 20, y se trata de lograr la mezcla perfecta con el nuevo fondo.
  • Tamaño de Erosión para Matizado Alfa: Esto ayuda a reducir un poco los bordes para que el recorte se vea más limpio. El tamaño predeterminado es 5, pero puedes ajustarlo para obtener el aspecto exacto que deseas.

Recuerda, no necesitas preocuparte por lograr todo perfecto. AiCasso es bastante bueno para entender lo que quieres decir, así que siéntete libre de ser tan casual o detallado como desees. ¡Feliz creación!
