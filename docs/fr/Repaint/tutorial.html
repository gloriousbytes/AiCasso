Astuces pour la Fonction Repeindre

Utiliser la fonction Inpaint dans AiCasso vous permet de modifier des zones spécifiques d'une image en les remplissant ou en les remplaçant. Que vous souhaitiez améliorer une image existante ou complètement altérer certaines sections, AiCasso rend le processus simple. Il vous suffit de sélectionner les zones que vous souhaitez changer, de fournir une invite, et de laisser AiCasso faire sa magie.

Options :

Force :

L'option de force contrôle combien de l'image originale est modifiée par l'invite. Elle varie de 0 à 1. Une valeur de 0 signifie que l'image originale est à peine altérée, tandis qu'une valeur de 1 signifie que l'image originale est presque complètement remplacée par les nouveaux détails que vous décrivez. Pensez-y comme à la 'force' de vos changements.

Invite 1 : "Une image hyper-réaliste d'un chat dans le parc."

Résultat : AiCasso générera un chat hyper-réaliste dans le parc, remplissant les zones peintes de manière transparente avec des textures et un éclairage réalistes.

Invite 2 : "Un chat dans le parc avec un style hyper-réaliste et une force de 0.7."

Résultat : Cela créera un chat hyper-réaliste dans le parc, avec la force de modification réglée à 0.7, intégrant les nouveaux éléments en douceur dans l'image originale.
