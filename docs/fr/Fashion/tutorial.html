Guide d'Utilisation de la Fonctionnalité de Génération de Mode

La fonctionnalité de Génération de Mode dans AiCasso vous permet d'habiller quelqu'un avec une tenue spécifique, comme une chemise, une robe ou un pantalon. Décrivez simplement ce que vous voulez, et regardez AiCasso appliquer le vêtement à la personne dans l'image.

Paramètres Optionnels

Si vous souhaitez ajouter votre propre style ou vous assurer que le résultat est parfait, vous pouvez ajuster ces paramètres :

  • Prompt Négatif : Si vous ne voulez pas quelque chose dans l'image, mentionnez-le ici. Laissez-le vide si tout ce que AiCasso propose vous convient.
  • Type de Vêtement : Indiquez à AiCasso si vous vous concentrez sur le haut du corps, le bas du corps, ou les robes. Si vous n'êtes pas sûr, le réglage par défaut est le haut du corps.
  • Nombre d'Étapes de Dénoyage : Cela détermine à quel point l'image est lisse et polie. Vous pouvez choisir entre 21, 31, ou 41 étapes, avec 21 comme valeur par défaut. Plus d'étapes signifient généralement un aspect plus raffiné.
  • Échelle de Guidance : Cela aide l'image à coller de près à votre description. Vous pouvez la régler de 1 à 20, avec 7 comme valeur par défaut. Des chiffres plus élevés signifient que AiCasso suivra votre prompt de plus près.

Rappelez-vous, ces paramètres sont optionnels ! AiCasso est assez intelligent et peut comprendre ce que vous voulez dire même si vous ne donnez pas tous les détails parfaitement. Amusez-vous et laissez libre cours à votre créativité !
