Guide d'utilisation de la fonctionnalité Image vers Vidéo

La fonctionnalité Image vers Vidéo dans AiCasso vous permet de transformer une seule image en une vidéo dynamique. Il vous suffit de décrire ce que vous souhaitez voir dans votre vidéo, et AiCasso s'en chargera !

Invite 1 :

"Un chiot mignon dans le parc"

Invite 2 :

Paramètres optionnels

Si vous souhaitez être plus précis avec votre vidéo, voici quelques paramètres que vous pouvez ajuster :

  • Invite négative : Décrivez ce que vous *ne* voulez *pas* dans la vidéo. Si laissé vide, AiCasso se concentrera uniquement sur l'invite positive.
  • Nombre d'images : Choisissez combien d'images (images individuelles) votre vidéo aura. Vous pouvez sélectionner jusqu'à 25 images, avec 16 comme valeur par défaut.
  • Étapes de débruitage : Cela contrôle la fluidité de votre vidéo. Plus il y a d'étapes, plus la vidéo est propre. Vous pouvez définir jusqu'à 50 étapes, avec 20 comme valeur par défaut.
  • Échelle de guidage : Cela aide la vidéo à rester proche de votre description. La plage va de 1.0 à 8.0, et vous pouvez ajuster les valeurs de début et de fin pour guider l'évolution de la vidéo.
  • Mouvement : Ajustez la quantité de mouvement dans votre vidéo. Une valeur plus élevée signifie plus d'action. La plage va de 0 à 127, avec 60 comme valeur par défaut.
  • Force d'augmentation du bruit : Ajoutez un peu de bruit à votre vidéo pour lui donner un aspect plus texturé. La plage va de 0 à 1, avec 0 comme valeur par défaut.
  • Images par seconde (FPS) : Définissez la vitesse de lecture de votre vidéo. La valeur par défaut est de 16 FPS, mais vous pouvez aller jusqu'à 16 pour une lecture plus rapide.
  • Méthode d'interpolation : Choisissez comment les images sont mélangées. La valeur par défaut est "lerp" (interpolation linéaire), mais vous pouvez également utiliser "slerp" (interpolation linéaire sphérique) pour différents effets.
  • Ignorer les clips : Ignorez certaines couches pour un style vidéo différent. Vous pouvez ignorer jusqu'à 2 couches, ou laisser nul pour le comportement par défaut.

Rappelez-vous, AiCasso est assez intelligent ! Vous n'avez pas besoin de rendre chaque détail parfait ou d'utiliser une grammaire correcte. Décrivez simplement ce que vous voulez, et AiCasso fera le reste. Amusez-vous à créer vos vidéos !
