Guide d'utilisation de la Fonction de Remplacement de Fond

La fonction de remplacement de fond dans AiCasso est parfaite lorsque vous souhaitez changer le fond de votre image. Que vous souhaitiez simplement supprimer le fond ou le remplacer par autre chose, AiCasso rend cela très facile. Vous n'avez pas à vous soucier des aspects techniques : décrivez simplement ce que vous voulez, et AiCasso s'occupera du reste.

Paramètres optionnels

Si vous souhaitez ajouter une touche personnelle ou affiner les résultats, voici quelques options avec lesquelles vous pouvez jouer :

  • Supprimer uniquement : Si vous souhaitez simplement supprimer le fond et ne pas le remplacer par quoi que ce soit, c'est votre option de prédilection. C'est comme appuyer sur un bouton de réinitialisation.
  • Matting alpha : C'est une façon élégante de dire "fondre les bords". Lorsque vous activez cela, cela aide à rendre les bords de votre sujet plus lisses et moins découpés. Par défaut, c'est désactivé.
  • Seuil de premier plan pour le matting alpha : Cela contrôle combien du sujet principal (comme une personne) reste visible lors de l'utilisation du matting alpha. La valeur par défaut est 240, et vous pouvez l'ajuster pour rendre les bords plus doux ou plus nets.
  • Seuil d'arrière-plan pour le matting alpha : Cela fait la même chose que le seuil de premier plan mais pour l'arrière-plan. La valeur par défaut est 20, et il s'agit d'obtenir le mélange parfait avec le nouveau fond.
  • Taille d'érosion pour le matting alpha : Cela aide à réduire un peu les bords pour rendre le découpage plus propre. La taille par défaut est 5, mais vous pouvez l'ajuster pour obtenir l'apparence exacte que vous souhaitez.

Rappelez-vous, vous n'avez pas besoin de vous stresser pour que tout soit parfait. AiCasso est assez bon pour comprendre ce que vous voulez dire, alors n'hésitez pas à être aussi décontracté ou détaillé que vous le souhaitez. Bonne création !
