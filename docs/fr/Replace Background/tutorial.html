Guide d'Utilisation de la Fonction Remplacer l'Arrière-plan

La fonction Remplacer l'Arrière-plan dans AiCasso est parfaite lorsque vous souhaitez changer l'arrière-plan de votre image. Que vous vouliez simplement supprimer l'arrière-plan ou le remplacer par autre chose, AiCasso rend cela très facile. Vous n'avez pas besoin de vous soucier des aspects techniques—décrivez simplement ce que vous voulez, et AiCasso fera le reste.

Paramètres Optionnels

Si vous souhaitez ajouter une touche personnelle ou affiner les résultats, voici quelques options avec lesquelles vous pouvez jouer :

  • Supprimer Seulement : Si vous souhaitez simplement supprimer l'arrière-plan sans le remplacer par quoi que ce soit, c'est votre option de prédilection. C'est comme appuyer sur un bouton de remise à zéro.
  • Matting Alpha : C'est une façon sophistiquée de dire "mélanger les bords". Lorsque vous l'activez, cela aide à rendre les bords de votre sujet plus lisses et moins découpés. Par défaut, il est désactivé.
  • Seuil de Premier Plan pour le Matting Alpha : Cela contrôle combien du sujet principal (comme une personne) reste visible lors de l'utilisation du matting alpha. La valeur par défaut est 240, et vous pouvez l'ajuster pour rendre les bords plus doux ou plus nets.
  • Seuil d'Arrière-plan pour le Matting Alpha : Cela fonctionne de la même manière que le seuil de premier plan mais pour l'arrière-plan. La valeur par défaut est 20, et il s'agit d'obtenir le mélange parfait avec le nouvel arrière-plan.
  • Taille d'Érosion pour le Matting Alpha : Cela aide à réduire un peu les bords pour que la découpe soit plus propre. La taille par défaut est 5, mais vous pouvez la modifier pour obtenir l'apparence exacte que vous souhaitez.

Rappelez-vous, vous n'avez pas besoin de vous stresser pour obtenir tout parfaitement. AiCasso est assez bon pour comprendre ce que vous voulez dire, alors n'hésitez pas à être aussi décontracté ou détaillé que vous le souhaitez. Bonne création !
