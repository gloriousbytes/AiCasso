Dicas de Repaint Feature

Usar o recurso Inpaint no AiCasso permite que você modifique áreas específicas de uma imagem preenchendo ou substituindo-as. Se você deseja melhorar uma imagem existente ou alterar completamente certas seções, o AiCasso torna o processo simples. Basta selecionar as áreas que deseja mudar, fornecer um prompt e deixar o AiCasso fazer sua mágica.

Opções:

Força:

A opção de força controla quanto da imagem original é alterada pelo prompt. Varia de 0 a 1. Um valor de 0 significa que a imagem original é pouco alterada, enquanto um valor de 1 significa que a imagem original é quase completamente substituída pelos novos detalhes que você descreve. Pense nisso como quão 'fortes' suas mudanças serão.

Estilo de Aprimoramento:

A opção de estilo de aprimoramento permite que você adicione um estilo artístico ou temático específico às áreas modificadas. Você pode escolher entre uma variedade de estilos, como:

Exemplos:

Prompt 1: "Uma imagem hiper-realista de um gato no parque."

Resultado: O AiCasso irá gerar um gato hiper-realista no parque, preenchendo as áreas pintadas de forma harmoniosa com texturas e iluminação realistas.

Prompt 2: "Um gato no parque com um estilo hiper-realista e uma força de 0.7."

Resultado: Isso criará um gato hiper-realista no parque, com a força de modificação definida para 0.7, misturando os novos elementos suavemente na imagem original.
