A função Substituir Fundo no AiCasso é perfeita para quando você deseja trocar o fundo da sua imagem. Seja para remover o fundo ou substituí-lo por outra coisa, o AiCasso torna isso super fácil. Você não precisa se preocupar com questões técnicas—basta descrever o que deseja, e o AiCasso fará o resto.

Parâmetros Opcionais

Se você quiser adicionar um toque pessoal ou ajustar os resultados, aqui estão algumas opções com as quais você pode brincar:

  • Remover Apenas: Se você deseja apenas remover o fundo e não substituí-lo por nada, esta é a sua opção. É como pressionar um botão de reiniciar.
  • Matização Alfa: Esta é uma maneira elegante de dizer "misturar as bordas." Quando você ativa isso, ajuda a deixar as bordas do seu sujeito mais suaves e menos recortadas. Por padrão, está desativado.
  • Limite de Primeiro Plano para Matização Alfa: Isso controla quanto do sujeito principal (como uma pessoa) permanece visível ao usar matização alfa. O padrão é 240, e você pode ajustá-lo para deixar as bordas mais suaves ou mais nítidas.
  • Limite de Fundo para Matização Alfa: Isso faz o mesmo que o limite de primeiro plano, mas para o fundo. O padrão é 20, e tudo se resume a conseguir a mistura perfeita com o novo fundo.
  • Tamanho de Erosão para Matização Alfa: Isso ajuda a reduzir um pouco as bordas para deixar o recorte mais limpo. O tamanho padrão é 5, mas você pode ajustá-lo para obter exatamente o visual que deseja.

Lembre-se, você não precisa se estressar para que tudo fique perfeito. O AiCasso é bastante bom em entender o que você quer dizer, então sinta-se à vontade para ser tão casual ou detalhado quanto desejar. Boa criação!
