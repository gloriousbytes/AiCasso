Umphumela 2:

Izinketho Ezingeziwe

Uma ufuna ukwengeza ukuthinta komuntu siqu noma ukuhlela imiphumela, nansi eminye imikhawulo ongayisebenzisa:

  • Susa Kuphela: Uma ufuna ukususa isizinda kuphela futhi ungafuni ukusishintsha ngento, le yinketho yakho. Kufana nokucindezela inkinobho yokuhlanza.
  • Alpha Matting: Le yindlela ethokozisayo yokuthi "hlanganisa imiphetho." Uma uyivula, izosiza ukwenza imiphetho yesikhumbuzo sakho ibonakale ibushelelezi futhi ingabi ne-cut-out. Ngokujwayelekile, ivaliwe.
  • Alpha Matting Foreground Threshold: Lokhu kulawula ukuthi mangaki emkhakheni omkhulu (njengomuntu) ohlala ubonakala uma usebenzisa i-alpha matting. Okujwayelekile ngu-240, futhi ungakwenza kube softer noma sharper.
  • Alpha Matting Background Threshold: Lokhu kwenza okufanayo njengomkhawulo we-foreground kodwa wesizinda. Okujwayelekile ngu-20, futhi konke kuhlobene nokuthola ukuhlanganiswa okufanele nesizinda esisha.
  • Alpha Matting Erode Size: Lokhu kusiza ukunciphisa imiphetho kancane ukuze i-cutout ibonakale ihlanzekile. Usayizi ojwayelekile ngu-5, kodwa ungawuhlela ukuze uthole ukubukeka okufanele.
