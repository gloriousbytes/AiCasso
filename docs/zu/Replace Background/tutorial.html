Umphumela 2:

Izinketho Ezingeziwe

Uma ufuna ukwengeza ukuthinta komuntu siqu noma ukulungisa imiphumela, nansi eminye imikhawulo ongayidlala nayo:

  • Susa Kuphela: Uma ufuna ukususa isizinda kuphela futhi ungafuni ukusishintsha ngento, le yinketho yakho. Kufana nokucindezela inkinobho yokuhlanzeka.
  • Alpha Matting: Le yindlela ethokozisayo yokuthi "hlanganisa imiphetho." Uma uyivula, kusiza ukwenza imiphetho yesihloko sakho ibonakale ibushelelezi futhi ingabi ne-cut-out. Ngokuvamile, ivaliwe.
  • Umkhawulo Wokuphakela we-Alpha Matting: Lokhu kulawula ukuthi mangaki amaphuzu wesihloko esikhulu (njengomuntu) ahlala ebukeka uma usebenzisa i-alpha matting. Okujwayelekile ngu-240, futhi ungakulungisa ukuze wenze imiphetho ibonakale ibushelelezi noma icace.
  • Umkhawulo Wesizinda se-Alpha Matting: Lokhu kwenza okufanayo nomkhawulo ophakeme kodwa wesizinda. Okujwayelekile ngu-20, futhi konke kuhlobene nokuthola ukuhlanganiswa okufanele nesizinda esisha.
  • Usayizi Wokuphonsa we-Alpha Matting: Lokhu kusiza ukunciphisa imiphetho kancane ukuze i-cutout ibonakale ihlanzekile. Usayizi ojwayelekile ngu-5, kodwa ungawulungisa ukuze uthole ukubukeka okufanele.

Khumbula, awudingi ukukhathazeka ngokuthola konke kuphumelele. I-AiCasso ikhululekile ekutholeni lokho okukhuluma ngakho, ngakho uzizwe ukhululekile ukuba ne-casual noma imininingwane njengoba uthanda. Jabulela ukudala!
