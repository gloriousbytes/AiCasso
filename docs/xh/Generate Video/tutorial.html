Iinketho Ezengeziwe

Ufuna ukulungisa iividiyo zakho? Nantsi eminye imigaqo onokuyijolisa:

  • Inani leefreyimu: Khetha ukuba zingaphi iifreyimu (imifanekiso eyodwa) iividiyo zakho ziza kuba nazo. Ungaya kwiifreyimu eziyi-25, kunye ne-16 njengezizathu ezisemthethweni.
  • Inani lezigaba zokunciphisa umsindo: Oku kulawula ukuba iividiyo zakho zibonakale njani. Izigaba ezininzi zithetha iividiyo ezithambileyo. Ungaseta ukuya kwi-50 zezigaba, kunye ne-20 njengezizathu ezisemthethweni.
  • Umgama Wokukhuthaza: Oku kunceda iividiyo ukuba zihlale zisondelelene nem描述 yakho. Umgama ophakamileyo, ngakumbi ulandela umphakamiso wakho. Umgama ungama-0 ukuya kwi-8, kunye ne-8 njengezizathu ezisemthethweni.
  • Ukuphosa Iklip: Phosa iilayini ezithile ukuze ufumane isitayela sevidiyo esahlukileyo. Ngokwemigaqo, akukho layini iphoswa, kodwa ungaphosa ukuya kwi-2.
  • Uphuculo Ububanzi kunye Nobubanzi: Ungayenza iividiyo zakho zibe zikhulu ngokulungisa umgangatho. Umgangatho osebenzayo ngu-1024x1024 pixels, kodwa ungayilungisa ukuze ihambelane neemfuno zakho.
  • Uphuculo Amandla: Oku kutshintsha ukuba iividiyo zakho ziguquka kangakanani xa zikhuliswa. Umgangatho osebenzayo ngu-1, kodwa ungayilungisa ukuze ubone umehluko omninzi.
  • Uphuculo Umgama Wokukhuthaza: Oku kukhokela inkqubo yokuphucula, enomgama ophakathi kuka-0 no-8, kunye ne-8 njengezizathu ezisemthethweni.
  • Uphuculo Izigaba Zokuphucula: Inani lezigaba ezithathiweyo ukuze kuphuculwe iividiyo. Izigaba ezininzi zenza ukuphucula kube smooth, kunye ne-maximum ye-50 kunye ne-20 njengezizathu ezisemthethweni.
  • Uphuculo Ukukhetha: Ufuna iividiyo ezichanekileyo? Vula oku. Iphosa ngokuzenzekelayo.
  • Uphuculo Ukukhetha Iseedi: Sebenzisa oku ukuze uvelise iividiyo ezifanayo ngexesha lonke ngokukhetha okuphuculweyo. Kukhetho kwaye iphoswe ngokuzenzekelayo.
  • Iifreyimu ngexesha (FPS): Beka isantya sevidiyo yakho. Umgangatho osebenzayo ngu-16 FPS, kodwa ungayilungisa ukuya kwi-16.
  • Imodeli yeVidiyo: Khetha isitayela sevidiyo yakho. Iinketho ziquka:
    • Izizathu ezisemthethweni: Imodeli yevidiyo eqhelekileyo.
    • I-Cartoon: Isitayela sevidiyo esifana ne-cartoon.
    • Imvelo: Isitayela sevidiyo esibukeka sikhululekile, semvelo.
    • I-Animated 3D: Isitayela sevidiyo esibonakalayo se-3D.

Khumbula, le migaqo ikhethiweyo. Umsebenzi Wokwenza Ividiyo usebenza kakuhle ngokuchaza nje, ngoko ke uzive ukulungelelanisa kwaye ubone ukuba ungayenza ntoni!
