Tipps zur Neuzeichnungsfunktion

Die Verwendung der Inpaint-Funktion in AiCasso ermöglicht es Ihnen, bestimmte Bereiche eines Bildes zu ändern, indem Sie sie ausfüllen oder ersetzen. Egal, ob Sie ein bestehendes Bild verbessern oder bestimmte Abschnitte vollständig ändern möchten, AiCasso macht den Prozess einfach. Wählen Sie einfach die Bereiche aus, die Sie ändern möchten, geben Sie einen Hinweis an und lassen Sie AiCasso seine Magie wirken.

Optionen:

Stärke:

Die Stärke-Option steuert, wie viel vom Originalbild durch den Hinweis geändert wird. Sie reicht von 0 bis 1. Ein Wert von 0 bedeutet, dass das Originalbild kaum verändert wird, während ein Wert von 1 bedeutet, dass das Originalbild fast vollständig durch die neuen Details ersetzt wird, die Sie beschreiben. Denken Sie daran, wie 'stark' Ihre Änderungen sein werden.

Stil verbessern:

Die Option Stil verbessern ermöglicht es Ihnen, einen bestimmten künstlerischen oder thematischen Stil zu den modifizierten Bereichen hinzuzufügen. Sie können aus einer Vielzahl von Stilen wählen, wie zum Beispiel:

Beispiele:

Hinweis 1: "Ein hyper-realistisches Bild einer Katze im Park."

Ergebnis: AiCasso wird eine hyper-realistischen Katze im Park generieren und die gemalten Bereiche nahtlos mit realistischen Texturen und Beleuchtung ausfüllen.

Hinweis 2: "Eine Katze im Park mit einem hyper-realistischen Stil und einer Stärke von 0,7."

Ergebnis: Dies wird eine hyper-realistischen Katze im Park erstellen, wobei die Änderungsstärke auf 0,7 eingestellt ist, und die neuen Elemente nahtlos in das Originalbild integriert werden.
