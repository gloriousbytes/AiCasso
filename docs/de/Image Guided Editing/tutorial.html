Tipps zur Bildgeführten Bearbeitung

Mit der Funktion "Bildgeführte Bearbeitung" in AiCasso können Sie Änderungen an einem Bild vornehmen, indem Sie einfach beschreiben, was Sie möchten. Egal, ob Sie den Stil ändern, die Stimmung anpassen oder bestimmte Teile bearbeiten möchten, AiCasso übernimmt das für Sie, ohne dass Sie manuell etwas auswählen oder malen müssen.

Wie es funktioniert:

Ausgangsbild:
Beschreiben Sie Ihre Änderung:

Geben Sie einfach ein, was Sie ändern möchten, und AiCasso wird diese Änderungen auf das Bild anwenden.

Beispielbearbeitungen:

Beispiel 1: "Mach sie blond."

Ergebnis: AiCasso wird die Haarfarbe der Person in blond ändern.

Beispiel 2: "Mach es Nacht, benutze 50 Schritte, eine Führungsskala von 1 und eine Bildführungsskala von 1."
Eingabebild:

Ergebnis: Das Bild wird in eine Nachtszene mit spezifischen Anpassungen basierend auf den von Ihnen bereitgestellten Details umgewandelt.

Zusätzliche Optionen

Sie können Ihre Bearbeitungen mit diesen optionalen Einstellungen feinabstimmen:

  • Bildführungsskala: Dies steuert, wie genau das neue Bild dem Original entspricht. Ein niedrigerer Wert bedeutet subtilere Änderungen, während ein höherer Wert sie stärker hervorhebt. Sie können diesen Wert zwischen 1 und 8 einstellen.
  • Schritte: Dies legt fest, wie viele Verfeinerungen das Tool vornimmt. Mehr Schritte bedeuten ein glatteres und detaillierteres Endbild. Sie können zwischen 1 und 50 Schritten wählen, wobei 20 der Standard ist.
  • Führungsskala: Dies beeinflusst, wie genau die KI Ihrer Beschreibung folgt. Eine niedrigere Skala gibt der KI mehr kreative Freiheit, während eine höhere Skala sie genau an das hält, was Sie gesagt haben. Sie können dies zwischen 1 und 20 anpassen.
