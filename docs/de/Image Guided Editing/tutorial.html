Tipps für bildgesteuertes Bearbeiten

Die Funktion für bildgesteuertes Bearbeiten in AiCasso ermöglicht es Ihnen, Änderungen an einem Bild vorzunehmen, indem Sie einfach beschreiben, was Sie möchten. Egal, ob Sie den Stil ändern, die Stimmung anpassen oder bestimmte Teile bearbeiten möchten, AiCasso erledigt das für Sie, ohne dass Sie manuell etwas auswählen oder malen müssen.

So funktioniert es:

Startbild:
Beschreiben Sie Ihre Änderung:

Geben Sie einfach ein, was Sie ändern möchten, und AiCasso wird diese Änderungen auf das Bild anwenden.

Beispieländerungen:

Beispiel 1: "Mach sie blond."

Ergebnis: AiCasso wird die Haarfarbe der Person auf blond ändern.

Beispiel 2: "Mach es Nacht, benutze 50 Schritte, eine Leitmaßstab von 1 und einen Bildleitmaßstab von 1."
Eingabebild:

Ergebnis: Das Bild wird in eine Nachtszene mit spezifischen Anpassungen basierend auf den von Ihnen bereitgestellten Details verwandelt.

Zusätzliche Optionen

Sie können Ihre Änderungen mit diesen optionalen Einstellungen verfeinern:

  • Bildleitmaßstab: Dies steuert, wie eng das neue Bild am Original haftet. Ein niedrigerer Wert bedeutet, dass die Änderungen subtiler sind, während ein höherer Wert sie auffälliger macht. Sie können diesen Wert zwischen 1 und 8 einstellen.
  • Schritte: Dies legt fest, wie viele Verfeinerungen das Tool vornehmen wird. Mehr Schritte bedeuten ein glatteres und detaillierteres Endbild. Sie können zwischen 1 und 50 Schritten wählen, wobei 20 der Standardwert ist.
  • Leitmaßstab: Dies beeinflusst, wie eng die KI Ihrer Beschreibung folgt. Ein niedrigerer Maßstab gibt der KI mehr kreative Freiheit, während ein höherer Maßstab sie dazu bringt, sich eng an das zu halten, was Sie gesagt haben. Sie können dies zwischen 1 und 20 anpassen.
