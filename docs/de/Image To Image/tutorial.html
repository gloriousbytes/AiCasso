Leitfaden zur Verwendung der Bild-zu-Bild-Funktion

Die Bild-zu-Bild-Funktion in AiCasso ist eine coole Möglichkeit, ein Bild, das Sie bereits haben, in etwas Neues zu verwandeln. Es ist wie eine Verwandlung Ihres Bildes basierend auf Ihrer eigenen Beschreibung!

So funktioniert es:

Eingabebild:
Prompt 1:

"Ein süßer Hund, benutze die Hälfte der Stärke"

Ergebnis 1:
Prompt 2:

"Eine süße 3D-CGI-Katze mit großen Augen, benutze 90% Stärke und den 3D-Cartoon-Stil"

Ergebnis 2:
Prompt 3:

"Eine schwarze Katze, mache sie realistisch"

Ergebnis 3:

Optionale Parameter

Wenn Sie Ihrer Bildtransformation eine persönliche Note verleihen möchten, finden Sie hier einige optionale Einstellungen, mit denen Sie experimentieren können. Aber keine Sorge – wenn Sie einfach nur beschreiben möchten, was Sie wollen, wird AiCasso die Arbeit ohne diese Extras erledigen!

  • Negativer Prompt: Wenn es etwas Bestimmtes gibt, das Sie nicht in Ihrem neuen Bild haben möchten, können Sie es hier erwähnen. Aber wenn Sie es leer lassen, ist das auch in Ordnung!
  • Stärke: Dies steuert, wie viel vom Originalbild verwendet wird. Sie können von 0,1 bis 1,0 gehen. Je niedriger die Zahl, desto mehr behält Ihr Bild sein ursprüngliches Aussehen. Der Standardwert ist 0,5, was ein gutes Gleichgewicht ist.
  • Stil verbessern: Möchten Sie Ihrem Bild eine bestimmte Stimmung verleihen? Wählen Sie einen Stil wie Comic, Chibi, 3D, Anime oder realistisch. Der Standard ist realistisch, aber fühlen Sie sich frei, zu experimentieren!

Und hey, denken Sie daran, dass Sie sich keine Sorgen machen müssen, alles perfekt zu machen. Beschreiben Sie einfach, was Sie in Ihren eigenen Worten möchten, und AiCasso erledigt den Rest. Viel Spaß beim Erstellen!
