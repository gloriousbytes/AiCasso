تصویر سے ویڈیو کی خصوصیت کے استعمال کی رہنمائی

AiCasso میں تصویر سے ویڈیو کی خصوصیت آپ کو ایک واحد تصویر کو متحرک ویڈیو میں تبدیل کرنے دیتی ہے۔ بس بیان کریں کہ آپ اپنی ویڈیو میں کیا دیکھنا چاہتے ہیں، اور AiCasso یہ کر دے گا!

نتیجہ 2:

اختیاری پیرامیٹرز

اگر آپ اپنی ویڈیو کے بارے میں مزید مخصوص ہونا چاہتے ہیں، تو یہاں کچھ سیٹنگز ہیں جنہیں آپ ایڈجسٹ کر سکتے ہیں:

  • منفی پرومپٹ: بیان کریں کہ آپ ویڈیو میں کیا *نہیں* چاہتے۔ اگر خالی چھوڑ دیا جائے تو، AiCasso صرف مثبت پرومپٹ پر توجہ دے گا۔
  • فریم کی تعداد: منتخب کریں کہ آپ کی ویڈیو میں کتنے فریم (انفرادی تصاویر) ہوں گے۔ آپ 25 فریم تک منتخب کر سکتے ہیں، جبکہ 16 ڈیفالٹ ہے۔
  • ڈی نوائزنگ کے مراحل: یہ کنٹرول کرتا ہے کہ آپ کی ویڈیو کتنی ہموار نظر آتی ہے۔ زیادہ مراحل کا مطلب ہے کہ ویڈیو زیادہ صاف ہے۔ آپ 50 مراحل تک سیٹ کر سکتے ہیں، جبکہ 20 ڈیفالٹ ہے۔
  • گائیڈنس اسکیل: یہ ویڈیو کو آپ کی وضاحت کے قریب رکھنے میں مدد کرتا ہے۔ یہ 1.0 سے 8.0 کے درمیان ہے، اور آپ ویڈیو کی ترقی کی رہنمائی کے لیے شروع اور اختتامی اقدار کو ایڈجسٹ کر سکتے ہیں۔
  • حرکت: اپنی ویڈیو میں حرکت کی مقدار کو ایڈجسٹ کریں۔ زیادہ قیمت کا مطلب ہے زیادہ عمل۔ یہ 0 سے 127 کے درمیان ہے، جبکہ 60 ڈیفالٹ ہے۔
  • شور کی بڑھتی ہوئی طاقت: اپنی ویڈیو میں کچھ شور شامل کریں تاکہ اسے زیادہ ساختی نظر آئے۔ یہ 0 سے 1 کے درمیان ہے، جبکہ 0 ڈیفالٹ ہے۔
  • فریم فی سیکنڈ (FPS): طے کریں کہ آپ کی ویڈیو کتنی تیزی سے چلتی ہے۔ ڈیفالٹ 16 FPS ہے، لیکن آپ تیز پلے بیک کے لیے 16 تک جا سکتے ہیں۔
  • انٹرپولیشن کا طریقہ: منتخب کریں کہ فریمز کو کس طرح ملایا جائے۔ ڈیفالٹ "لرپ" (لکیری انٹرپولیشن) ہے، لیکن آپ مختلف اثرات کے لیے "سلرپ" (سفیری لکیری انٹرپولیشن) بھی استعمال کر سکتے ہیں۔
  • کلپ اسکیپ: مختلف ویڈیو طرز کے لیے کچھ تہوں کو چھوڑ دیں۔ آپ 2 تہوں تک چھوڑ سکتے ہیں، یا ڈیفالٹ رویے کے لیے اسے خالی چھوڑ سکتے ہیں۔

یاد رکھیں، AiCasso کافی ذہین ہے! آپ کو ہر تفصیل کو درست کرنے یا صحیح گرامر استعمال کرنے کی ضرورت نہیں ہے۔ بس بیان کریں کہ آپ کیا چاہتے ہیں، اور AiCasso باقی کرے گا۔ اپنی ویڈیوز بنانے میں مزہ کریں!
