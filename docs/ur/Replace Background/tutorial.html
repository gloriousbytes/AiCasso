پس منظر تبدیل کرنے کی خصوصیت کے استعمال کی رہنمائی

AiCasso میں پس منظر تبدیل کرنے کی خصوصیت اس وقت کے لیے بہترین ہے جب آپ اپنی تصویر کا پس منظر تبدیل کرنا چاہتے ہیں۔ چاہے آپ صرف پس منظر کو ہٹانا چاہتے ہوں یا اسے کسی اور چیز سے تبدیل کرنا چاہتے ہوں، AiCasso یہ سب کچھ بہت آسان بنا دیتا ہے۔ آپ کو تکنیکی چیزوں کی فکر کرنے کی ضرورت نہیں—بس بتائیں کہ آپ کیا چاہتے ہیں، اور AiCasso باقی سب کچھ کرے گا۔

یہ کیسے کام کرتا ہے:

پرومپٹ 1:

"پس منظر ہٹائیں"

پرومپٹ 2:

"پس منظر کو جنگل سے تبدیل کریں، صرف ہٹانے کے بجائے، الفا میٹنگ بند کریں۔ سامنے کی تھریشولڈ 240، پس منظر کی تھریشولڈ 20، اور الفا میٹنگ کے لیے ایروڈ سائز 5 استعمال کریں۔"

نتیجہ 2:

اختیاری پیرامیٹرز

اگر آپ ذاتی ٹچ شامل کرنا چاہتے ہیں یا نتائج کو بہتر بنانا چاہتے ہیں، تو یہاں کچھ اختیارات ہیں جن کے ساتھ آپ کھیل سکتے ہیں:

  • صرف ہٹائیں: اگر آپ صرف پس منظر کو ہٹانا چاہتے ہیں اور اسے کسی چیز سے تبدیل نہیں کرنا چاہتے، تو یہ آپ کا بہترین انتخاب ہے۔ یہ ایک صاف سلیٹ بٹن دبانے کی طرح ہے۔
  • الفا میٹنگ: یہ "کناروں کو ملا دیں" کہنے کا ایک شاندار طریقہ ہے۔ جب آپ اسے آن کرتے ہیں، تو یہ آپ کے موضوع کے کناروں کو ہموار اور کم کٹ آؤٹ دکھانے میں مدد کرتا ہے۔ ڈیفالٹ کے طور پر، یہ بند ہے۔
  • الفا میٹنگ سامنے کی تھریشولڈ: یہ کنٹرول کرتا ہے کہ مرکزی موضوع (جیسے ایک شخص) کتنی مقدار میں نظر آتا ہے جب الفا میٹنگ استعمال کی جاتی ہے۔ ڈیفالٹ 240 ہے، اور آپ اسے کناروں کو نرم یا تیز کرنے کے لیے ایڈجسٹ کر سکتے ہیں۔
  • الفا میٹنگ پس منظر کی تھریشولڈ: یہ سامنے کی تھریشولڈ کی طرح ہی کام کرتا ہے لیکن پس منظر کے لیے۔ ڈیفالٹ 20 ہے، اور یہ نئے پس منظر کے ساتھ بہترین ملاپ حاصل کرنے کے بارے میں ہے۔
  • الفا میٹنگ ایروڈ سائز: یہ کناروں کو تھوڑا سکڑنے میں مدد کرتا ہے تاکہ کٹ آؤٹ زیادہ صاف نظر آئے۔ ڈیفالٹ سائز 5 ہے، لیکن آپ اسے اپنی مطلوبہ شکل حاصل کرنے کے لیے ایڈجسٹ کر سکتے ہیں۔

یاد رکھیں، آپ کو ہر چیز کو مکمل کرنے کی فکر کرنے کی ضرورت نہیں۔ AiCasso آپ کے مطلب کو سمجھنے میں کافی اچھا ہے، لہذا آپ جتنا چاہیں آرام دہ یا تفصیلی ہو سکتے ہیں۔ تخلیق کرنے میں خوش رہیں!
