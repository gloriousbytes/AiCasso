پس منظر تبدیل کرنے کی خصوصیت کے استعمال کی رہنمائی

AiCasso میں پس منظر تبدیل کرنے کی خصوصیت آپ کی تصویر میں پس منظر کو تبدیل کرنے کے لئے بہترین ہے۔ چاہے آپ صرف پس منظر کو ہٹانا چاہتے ہوں یا اسے کچھ اور سے تبدیل کرنا چاہتے ہوں، AiCasso یہ بہت آسان بناتا ہے۔ آپ کو تکنیکی چیزوں کی فکر کرنے کی ضرورت نہیں ہے—بس یہ بیان کریں کہ آپ کیا چاہتے ہیں، اور AiCasso باقی کام کرے گا۔

پرومپٹ 2:

نتیجہ 2:

اختیاری پیرامیٹرز

اگر آپ ذاتی ٹچ شامل کرنا چاہتے ہیں یا نتائج کو بہتر بنانا چاہتے ہیں، تو یہاں کچھ اختیارات ہیں جن کے ساتھ آپ کھیل سکتے ہیں:

  • صرف ہٹائیں: اگر آپ صرف پس منظر کو ہٹانا چاہتے ہیں اور اسے کسی چیز سے تبدیل نہیں کرنا چاہتے، تو یہ آپ کا بہترین انتخاب ہے۔ یہ ایک صاف سلیٹ بٹن دبانے کی طرح ہے۔
  • الفا میٹنگ: یہ "کناروں کو ملا دیں" کہنے کا ایک شاندار طریقہ ہے۔ جب آپ اسے آن کرتے ہیں، تو یہ آپ کے موضوع کے کناروں کو ہموار اور کم کٹ آؤٹ دکھانے میں مدد کرتا ہے۔ ڈیفالٹ کے طور پر، یہ بند ہے۔
  • الفا میٹنگ سامنے کا تھریشولڈ: یہ کنٹرول کرتا ہے کہ جب الفا میٹنگ کا استعمال کیا جائے تو مرکزی موضوع (جیسے ایک شخص) کتنا نظر آتا ہے۔ ڈیفالٹ 240 ہے، اور آپ اسے کناروں کو نرم یا تیز کرنے کے لئے ایڈجسٹ کر سکتے ہیں۔
  • الفا میٹنگ پس منظر کا تھریشولڈ: یہ سامنے کے تھریشولڈ کی طرح ہی کام کرتا ہے لیکن پس منظر کے لئے۔ ڈیفالٹ 20 ہے، اور یہ نئے پس منظر کے ساتھ بہترین ملاپ حاصل کرنے کے بارے میں ہے۔
  • الفا میٹنگ ایروڈ سائز: یہ کناروں کو تھوڑا سا سکڑنے میں مدد کرتا ہے تاکہ کٹ آؤٹ زیادہ صاف نظر آئے۔ ڈیفالٹ سائز 5 ہے، لیکن آپ اسے اپنی مطلوبہ شکل حاصل کرنے کے لئے ایڈجسٹ کر سکتے ہیں۔

یاد رکھیں، آپ کو ہر چیز کو مکمل کرنے کے بارے میں فکر کرنے کی ضرورت نہیں ہے۔ AiCasso آپ کے مطلب کو سمجھنے میں کافی اچھا ہے، لہذا آپ جتنا چاہیں آرام دہ یا تفصیلی ہو سکتے ہیں۔ تخلیق کا لطف اٹھائیں!
