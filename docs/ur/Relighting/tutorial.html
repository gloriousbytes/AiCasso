اختیاری تبدیلیاں

اگر آپ مہم جوئی محسوس کر رہے ہیں اور اپنی ری لائٹنگ میں تھوڑی مزید چمک شامل کرنا چاہتے ہیں، تو یہاں کچھ چیزیں ہیں جنہیں آپ تبدیل کر سکتے ہیں:

  • منفی پرامپٹ: کیا آپ کچھ سے بچنا چاہتے ہیں؟ آپ AiCasso کو بتا سکتے ہیں کہ کیا شامل نہ کریں۔ لیکن فکر نہ کریں، اگر آپ اسے خالی چھوڑ دیتے ہیں تو AiCasso صرف اس پر توجہ دے گا جو آپ چاہتے ہیں۔
  • روشنی کی سمت: آپ یہ بتا سکتے ہیں کہ روشنی کہاں سے آنی چاہیے—بائیں، دائیں، اوپر، نیچے، یا بالکل بھی نہیں۔ ڈیفالٹ کے طور پر، روشنی بائیں سے آتی ہے۔
  • گائیڈنس اسکیل: یہ AiCasso کو آپ کی وضاحت پر قائم رہنے میں مدد کرتا ہے۔ اسکیل 1 سے 20 تک ہے، جس میں 7 ڈیفالٹ ہے۔ جتنا زیادہ نمبر ہوگا، اتنا ہی AiCasso آپ کے پرامپٹ پر قائم رہے گا۔
  • ہائی-ریزولوشن اسکیل: کیا آپ چاہتے ہیں کہ آپ کی تصویر واضح ہو؟ یہ وضاحت کو کنٹرول کرتا ہے۔ ڈیفالٹ 1.5 ہے، لیکن آپ اسے 1.0 سے 2.0 تک سیٹ کر سکتے ہیں تاکہ تصویر زیادہ واضح ہو۔
  • ڈی نوائز لیولز: دو سطحیں ہیں—کم ریزولوشن اور ہائی ریزولوشن۔ یہ کنٹرول کرتے ہیں کہ آخری تصویر کتنی ہموار یا تفصیلی نظر آتی ہے۔ آپ کم ریزولوشن ڈی نوائز لیول کو 0.0 اور 1.0 کے درمیان سیٹ کر سکتے ہیں (ڈیفالٹ 0.9 ہے) اور ہائی ریزولوشن کو 0.0 اور 1.0 کے درمیان (ڈیفالٹ 0.5 ہے)۔
  • قدموں کی تعداد: یہ وہ تعداد ہے جس میں AiCasso آپ کی تصویر کو بہتر بناتا ہے۔ ڈیفالٹ 50 ہے، لیکن آپ 20 سے 50 قدموں تک جا سکتے ہیں۔
