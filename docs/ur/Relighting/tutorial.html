  • رہنمائی کا پیمانہ: یہ AiCasso کو آپ کی وضاحت پر قائم رہنے میں مدد کرتا ہے۔ یہ پیمانہ 1 سے 20 تک ہے، جس میں 7 ڈیفالٹ ہے۔ جتنا زیادہ نمبر ہوگا، اتنا ہی AiCasso آپ کے پرامپٹ پر قائم رہے گا۔
  • ہائی ریزولوشن اسکیل: کیا آپ چاہتے ہیں کہ آپ کی تصویر نمایاں ہو؟ یہ وضاحت کو کنٹرول کرتا ہے۔ ڈیفالٹ 1.5 ہے، لیکن آپ اسے 1.0 سے 2.0 تک سیٹ کر سکتے ہیں تاکہ تصویر زیادہ واضح ہو۔
  • ڈی نوائز لیولز: دو سطحیں ہیں—کم ریزولوشن اور ہائی ریزولوشن۔ یہ کنٹرول کرتے ہیں کہ آخری تصویر کتنی ہموار یا تفصیلی نظر آتی ہے۔ آپ کم ریزولوشن ڈی نوائز لیول کو 0.0 سے 1.0 (ڈیفالٹ 0.9) اور ہائی ریزولوشن کو 0.0 سے 1.0 (ڈیفالٹ 0.5) کے درمیان سیٹ کر سکتے ہیں۔
  • قدموں کی تعداد: یہ ہے کہ AiCasso آپ کی تصویر کو پالش کرنے کے لیے کتنے قدم اٹھاتا ہے۔ ڈیفالٹ 50 ہے، لیکن آپ 20 سے 50 قدموں تک جا سکتے ہیں۔
