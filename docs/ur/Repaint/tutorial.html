ری پینٹ فیچر کے نکات

AiCasso میں ان پینٹ فیچر کا استعمال آپ کو ایک تصویر کے مخصوص علاقوں میں تبدیلی کرنے کی اجازت دیتا ہے، چاہے آپ موجودہ تصویر کو بہتر بنانا چاہتے ہوں یا کچھ حصوں کو مکمل طور پر تبدیل کرنا چاہتے ہوں، AiCasso یہ عمل آسان بناتا ہے۔ بس ان علاقوں کا انتخاب کریں جنہیں آپ تبدیل کرنا چاہتے ہیں، ایک پرامپٹ فراہم کریں، اور AiCasso کو اپنا جادو کرنے دیں۔

اختیارات:

طاقت:

طاقت کا اختیار کنٹرول کرتا ہے کہ پرامپٹ کے ذریعے اصل تصویر میں کتنی تبدیلی کی جائے گی۔ یہ 0 سے 1 تک ہوتا ہے۔ 0 کی قیمت کا مطلب ہے کہ اصل تصویر میں بہت کم تبدیلی کی گئی ہے، جبکہ 1 کی قیمت کا مطلب ہے کہ اصل تصویر تقریباً مکمل طور پر آپ کی بیان کردہ نئی تفصیلات سے تبدیل ہو گئی ہے۔ اسے اس طرح سوچیں کہ آپ کی تبدیلیاں کتنی 'مضبوط' ہوں گی۔

اسٹائل کو بہتر بنائیں:

اسٹائل کو بہتر بنانے کا اختیار آپ کو تبدیل شدہ علاقوں میں ایک مخصوص فنکارانہ یا تھیماتی اسٹائل شامل کرنے کی اجازت دیتا ہے۔ آپ مختلف اسٹائلز میں سے انتخاب کر سکتے ہیں جیسے:

یہ کیسے کام کرتا ہے:

ان پٹ تصویر:
ان علاقوں کو پینٹ کریں جنہیں بھرنا یا تبدیل کرنا ہے:

مثالیں:

پرامپٹ 1: "پارک میں بلی کی ایک ہائپر ریئلسٹک تصویر۔"

نتیجہ: AiCasso پارک میں ایک ہائپر ریئلسٹک بلی تیار کرے گا، پینٹ کیے گئے علاقوں کو حقیقت پسندانہ ساختوں اور روشنی کے ساتھ بغیر کسی رکاوٹ کے بھر دے گا۔

پرامپٹ 2: "پارک میں ایک بلی جس کا ہائپر ریئلسٹک اسٹائل اور طاقت 0.7 ہے۔"

نتیجہ: یہ پارک میں ایک ہائپر ریئلسٹک بلی بنائے گا، جس میں تبدیلی کی طاقت 0.7 پر سیٹ کی گئی ہے، نئی عناصر کو اصل تصویر میں ہموار طریقے سے ضم کرتا ہے۔

  • پکسل آرٹ: ایک ریٹرو، ویڈیو گیم سے متاثرہ شکل تخلیق کرنے کے لیے پکسل شدہ ڈیزائن کا استعمال کرتا ہے۔
  • ساخت: بصری گہرائی کو بڑھانے کے لیے امیر، تفصیلی ساختوں پر توجہ مرکوز کرتا ہے۔
