ری پینٹ فیچر کی تجاویز

AiCasso میں ان پینٹ فیچر کا استعمال آپ کو کسی تصویر کے مخصوص علاقوں میں تبدیلی کرنے کی اجازت دیتا ہے۔ چاہے آپ موجودہ تصویر کو بہتر بنانا چاہتے ہوں یا کچھ حصوں کو مکمل طور پر تبدیل کرنا چاہتے ہوں، AiCasso یہ عمل آسان بناتا ہے۔ بس ان علاقوں کا انتخاب کریں جنہیں آپ تبدیل کرنا چاہتے ہیں، ایک پرامپٹ فراہم کریں، اور AiCasso کو اپنا جادو کرنے دیں۔

اختیارات:

طاقت:

طاقت کا اختیار کنٹرول کرتا ہے کہ پرامپٹ کے ذریعے اصل تصویر میں کتنا تبدیلی کی جائے گی۔ یہ 0 سے 1 کے درمیان ہوتا ہے۔ 0 کی قیمت کا مطلب ہے کہ اصل تصویر میں بہت کم تبدیلی کی گئی ہے، جبکہ 1 کی قیمت کا مطلب ہے کہ اصل تصویر تقریباً مکمل طور پر نئی تفصیلات سے تبدیل ہو گئی ہے۔ اسے اس طرح سمجھیں کہ آپ کی تبدیلیاں کتنی 'مضبوط' ہوں گی۔

اسٹائل کو بہتر بنائیں:

اسٹائل کو بہتر بنانے کا اختیار آپ کو تبدیل شدہ علاقوں میں مخصوص فنکارانہ یا تھیماتی اسٹائل شامل کرنے کی اجازت دیتا ہے۔ آپ مختلف اسٹائلز میں سے انتخاب کر سکتے ہیں جیسے:

یہ کیسے کام کرتا ہے:

ان پٹ امیج:
پر رنگ بھرنے یا تبدیل کرنے کے لئے علاقوں کو پینٹ کریں:

مثالیں:

پرامپٹ 1: "پارک میں ایک ہائپر ریئلسٹک بلی کی تصویر۔"

نتیجہ: AiCasso پارک میں ایک ہائپر ریئلسٹک بلی پیدا کرے گا، پینٹ کیے گئے علاقوں کو حقیقت پسندانہ ٹیکسچر اور روشنی کے ساتھ بھر دے گا۔

پرامپٹ 2: "پارک میں ایک بلی ہائپر ریئلسٹک اسٹائل کے ساتھ اور طاقت 0.7۔"

نتیجہ: یہ پارک میں ایک ہائپر ریئلسٹک بلی بنائے گا، جس میں تبدیلی کی طاقت 0.7 پر سیٹ کی گئی ہے، نئی عناصر کو اصل تصویر میں ہموار طریقے سے ملاتے ہوئے۔

  • ساخت: بصری گہرائی کو بڑھانے کے لئے امیر، تفصیلی ساختوں پر توجہ مرکوز کرتا ہے۔
