AiCassoのプライバシーポリシー

発効日: 2024年9月13日

AiCassoでは、あなたのプライバシーを最優先に考えています。このプライバシーポリシーは、私たちのアプリケーションを使用する際に提供される情報の取り扱いについて説明します。

  1. 収集する情報
    • 入力画像: AiCassoにアップロードされた画像は、サービス提供のためのみに収集されます。
    • 顔データ: 顔が含まれる画像をアップロードした場合、ユーザーが選択した特徴に基づいてカスタマイズされた画像を生成するために顔の特徴を処理します。この顔データは、アップロードされた画像から抽出され、画像生成の目的のみに使用されます。
  2. 情報の使用方法
    • サービス提供: 入力画像（顔データを含む）は、AiCassoが提供する機能を実行するためのみに使用されます。これには、選択した特徴に基づいて新しい画像を生成することが含まれます。
  3. データの保持
    • アップロードされた画像の即時削除: すべての入力画像は、処理が完了し、その目的が達成された後に即座に削除されます。
    • 生成された画像の一時保存: 生成された画像は、私たちのクラウドサービスに保存され、ユーザーは24時間利用可能です。この期間が過ぎると、これらの画像は自動的に削除されます。
    • クラウドストレージ管理: 私たちのクラウドストレージは、残留データが残らないように24時間ごとにクリアされます。
  4. データの共有
    • 第三者への共有なし: 私たちは、あなたの画像、顔データ、または個人データを第三者と共有、販売、または開示することはありません。すべてのデータ処理は、AiCasso内で内部的に処理されます。
  5. セキュリティ対策
    • データ保護: 私たちは、処理中およびクラウドサービスに保存中のあなたの画像と顔データを保護するために、業界標準のセキュリティ対策を講じています。
  6. このプライバシーポリシーの変更
    • ポリシーの更新: 私たちは、このプライバシーポリシーを随時更新することがあります。変更があった場合は、更新された発効日が反映されます。
