ပြန်ဆင်ခြင်း အင်္ဂါရပ် အကြံပြုချက်များ

AiCasso တွင် Inpaint အင်္ဂါရပ်ကို အသုံးပြုခြင်းသည် ပုံတစ်ပုံ၏ အထူးနေရာများကို ဖြည့်စွက်ခြင်း သို့မဟုတ် အစားထိုးခြင်းဖြင့် ပြင်ဆင်နိုင်သည်။ သင်သည် ရှိပြီးသား ပုံကို တိုးတက်စေလိုပါက သို့မဟုတ် အပိုင်းအချို့ကို အပြောင်းအလဲ လုပ်လိုပါက AiCasso သည် လုပ်ငန်းစဉ်ကို ရိုးရှင်းစေသည်။ သင်ပြောင်းလဲလိုသော နေရာများကို ရွေးချယ်ပြီး အကြောင်းအရာတစ်ခု ပေးပါ၊ AiCasso သည် ၎င်း၏ မက်ဂျစ်ကို လုပ်ဆောင်ပါလိမ့်မည်။

ရွေးချယ်စရာများ:

အားသာချက်:

အားသာချက်ရွေးချယ်မှုသည် အကြောင်းအရာမှ ပြောင်းလဲမှုကို ထိန်းချုပ်သည်။ ၎င်းသည် 0 မှ 1 အထိ ရှိသည်။ 0 သည် မူရင်းပုံကို အနည်းငယ်သာ ပြောင်းလဲသည်ကို ဆိုလိုပြီး 1 သည် မူရင်းပုံကို သင်ဖော်ပြသော အသစ်သော အသေးစိတ်များဖြင့် အပြောင်းအလဲ လုပ်ထားသည်ကို ဆိုလိုသည်။ သင်၏ ပြောင်းလဲမှုများသည် 'အားသာ' ဖြစ်မည်ကို စဉ်းစားပါ။

စတိုင် တိုးတက်စေခြင်းရွေးချယ်မှုသည် ပြင်ဆင်ထားသော နေရာများတွင် အထူး အနုပညာ သို့မဟုတ် သီအိုရီစတိုင်တစ်ခု ထည့်သွင်းရန် ခွင့်ပြုသည်။ သင်သည် အမျိုးမျိုးသော စတိုင်များမှ ရွေးချယ်နိုင်သည်။

အကြောင်းအရာ ၂: "ဥယျာဉ်တွင် ကြောင်တစ်ကောင်ကို အထူးအထင်ရှားသော စတိုင်နှင့် 0.7 အားသာချက်ဖြင့်။"
