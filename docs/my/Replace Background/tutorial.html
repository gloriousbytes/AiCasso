နောက်ခံအစားထိုးခြင်း လမ်းညွှန်

AiCasso တွင် နောက်ခံအစားထိုးခြင်း လုပ်ဆောင်ချက်သည် သင်၏ပုံတွင် နောက်ခံကို အစားထိုးလိုသောအခါ အထူးသင့်လျော်သည်။ သင်သည် နောက်ခံကို ဖျက်လိုပါက သို့မဟုတ် အခြားအရာတစ်ခုဖြင့် အစားထိုးလိုပါက AiCasso သည် အလွန်လွယ်ကူစေသည်။ နည်းပညာဆိုင်ရာအရာများကို စိုးရိမ်စရာမလိုပါ—သင်လိုချင်သောအရာကို ဖေါ်ပြပါ၊ AiCasso သည် အခြားအရာများကို လုပ်ဆောင်ပါလိမ့်မည်။

ဒါက ဘယ်လိုလုပ်ဆောင်သလဲ:

ပုံထည့်ခြင်း:
အကြံပြုချက် ၁:

"နောက်ခံကို ဖျက်ပါ"

ရလဒ် ၁:
အကြံပြုချက် ၂:

"နောက်ခံကို သစ်တောဖြင့် အစားထိုးပါ၊ ဖျက်ခြင်းသာမဟုတ်ဘဲ၊ alpha matting ကို ပိတ်ထားပါ။ Foreground threshold ကို 240၊ background threshold ကို 20 နှင့် alpha matting အတွက် erode size ကို 5 သတ်မှတ်ပါ။"

ရလဒ် ၂:

ရွေးချယ်စရာ အချက်အလက်များ

သင်သည် ကိုယ်ပိုင်ထိပ်ထားမှုတစ်ခု ထည့်သွင်းလိုပါက သို့မဟုတ် ရလဒ်များကို လိုက်လျောညီထွေစေရန် အချို့သောရွေးချယ်စရာများကို စမ်းသပ်နိုင်ပါသည်။

  • ဖျက်ခြင်းသာ: သင်သည် နောက်ခံကို ဖျက်လိုသာဖြစ်ပြီး အခြားအရာဖြင့် အစားထိုးလိုမဟုတ်ပါက၊ ဤသည် သင်၏ ရွေးချယ်စရာဖြစ်သည်။ ၎င်းသည် သန့်ရှင်းသော စာမျက်နှာကို နှိပ်ခြင်းနှင့် တူသည်။
  • Alpha Matting: ၎င်းသည် "အရိပ်များကို ပေါင်းစပ်ပါ" ဟု ဆိုသော စကားလုံးဖြစ်သည်။ သင်သည် ဤကို ဖွင့်ပါက သင်၏အကြောင်းအရာ၏ အရိပ်များကို ပိုမိုချောမွေ့စေပြီး ကွဲပြားမှုကို လျှော့ချနိုင်သည်။ အခြေခံအားဖြင့် ၎င်းသည် ပိတ်ထားသည်။
  • Alpha Matting Foreground Threshold: ဤသည်သည် alpha matting ကို အသုံးပြုသောအခါ အဓိပ္ပါယ်ရှိသော အကြောင်းအရာ (လူတစ်ယောက်ကဲ့သို့) မည်မျှမြင်ရမည်ကို ထိန်းချုပ်သည်။ အခြေခံအားဖြင့် 240 ဖြစ်ပြီး သင်သည် အရိပ်များကို ပိုမိုချောမွေ့စေသို့မဟုတ် ပိုမိုရှင်းလင်းစေရန် ပြင်ဆင်နိုင်သည်။
  • Alpha Matting Background Threshold: ဤသည်သည် foreground threshold နှင့် တူညီသည်၊ သို့သော် နောက်ခံအတွက်ဖြစ်သည်။ အခြေခံအားဖြင့် 20 ဖြစ်ပြီး သစ်တောအသစ်နှင့် အထူးပေါင်းစပ်မှုကို ရရှိရန် အရေးကြီးသည်။
  • Alpha Matting Erode Size: ဤသည်သည် အရိပ်များကို အနည်းငယ် လျှော့ချရန် ကူညီသည်။ အခြေခံအားဖြင့် 5 ဖြစ်ပြီး သင်လိုချင်သော ပုံစံကို ရရှိရန် ပြင်ဆင်နိုင်သည်။

သင်သည် အရာအားလုံးကို အထူးပြုရန် စိုးရိမ်စရာမလိုပါ။ AiCasso သည် သင်၏ အဓိပ္ပါယ်ကို နားလည်ရန် အထူးကောင်းမွန်သည်၊ ထို့ကြောင့် သင်သည် အဆင်ပြေစွာ သို့မဟုတ် အသေးစိတ်ဖြင့် ဖျော်ဖြေရန် လွတ်လပ်ပါ။ ဖန်တီးမှုကို ပျော်ပါစေ!
