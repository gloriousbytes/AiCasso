Tips för att använda Outpaint-funktionen

Outpaint-funktionen i AiCasso låter dig utöka en bild genom att lägga till nya element runt omkring eller förlänga det som redan finns. Beskriv bara vad du vill ha, så tar AiCasso hand om resten.

Beskriv din förändring:

Du skriver bara vad du vill lägga till eller utöka, så tillämpar AiCasso dessa förändringar. Du kan specificera riktningar som vänster, höger, upp eller ner, och till och med detaljer som att zooma ut eller panorera.

Exempel på redigeringar:

Exempel 1: "Lägg till ett isberg till höger med en översättningsfaktor på 0.4."

Resultat: AiCasso utökar bilden åt höger och lägger till ett isberg, vilket håller det konsekvent med den befintliga scenen.

Exempel 2: "Zooma ut, behåll konsistensen av det mörka havet på natten."

Resultat: Bilden zoomar ut och avslöjar mer av det mörka havet, samtidigt som atmosfären av nattscenen bibehålls.

Ytterligare alternativ

Här är några valfria inställningar som du kan använda för att finjustera dina redigeringar:

  • Negativ prompt: Om det finns något specifikt du inte vill ha i bilden kan du berätta för AiCasso här. Till exempel kan du vilja undvika att lägga till vissa färger eller objekt.
  • Gångtyp: Detta avgör riktningen i vilken bilden expanderar. Du kan välja mellan vänster, höger, upp, ner eller bakåt (vilket zoomar ut). Till exempel, för att panorera åt höger, skulle du välja 'höger'.
  • Översättningsfaktor: Detta kontrollerar hur mycket bilden rör sig med varje steg när den expanderar. Ett högre värde flyttar bilden mer och avslöjar mer av det nya innehållet. Standard är 0.05, men du kan justera det från 0 till 1.
  • Bredd/Höjd översättning per steg: Dessa kontrollerar hur långt bilden rör sig horisontellt (bredd) eller vertikalt (höjd) med varje steg. Standard är 32 pixlar per steg, men du kan justera detta baserat på hur mycket du vill att bilden ska flytta sig.
  • Antal inferens- eller avbrusningssteg: Detta avgör hur många justeringar AiCasso gör för att få den slutliga bilden. Fler steg leder till ett jämnare resultat, med en standard på 20 steg, men du kan gå upp till 50.
  • Antal interpolationssteg: Detta kontrollerar hur smidigt övergången sker mellan två riktningar i en enda expansion. Standard är 60, men du kan öka det upp till 120 för smidigare övergångar.

Kom ihåg, dessa inställningar är valfria. AiCasso är utformat för att fungera bra med bara din naturliga språkliga beskrivning. Känn dig fri att experimentera och se vad som fungerar bäst för din bild!
