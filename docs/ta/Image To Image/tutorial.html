  • Fantasy Art: கற்பனை கலைக்கான அடிப்படையில் விவரமான, கற்பனை காட்சிகளை வலுப்படுத்துகிறது.
  • Isometric: மூன்று பரிமாண தோற்றத்தை உருவாக்க ஒரு ஐசோமெட்ரிக் பார்வையைப் பயன்படுத்துகிறது.
  • Line Art: விவரமான வரைபடங்களை உருவாக்க சுத்தமான, தெளிவான வரிகளை மையமாகக் கொண்டுள்ளது.
  • Low Poly: குறைந்த-பொலிகான் 3D மாதிரிகளைப் பயன்படுத்தி ஒரு எளிமையான, கோண தோற்றத்தை உருவாக்குகிறது.
  • Neonpunk: நியான் ஒளியுடன் கூடிய சைபர்பங்க் அழகியைக் இணைக்கிறது.
  • Origami: ஒரிகாமியின் அடிப்படையில் சிக்கலான, மடிக்கோல் வடிவங்களை மையமாகக் கொண்டுள்ளது.
  • Photographic: யதார்த்தமான, உயிரோட்டமான காட்சிகளை நோக்குகிறது.
  • Pixel Art: பழமையான, வீடியோ விளையாட்டு-ஊக்கமளிக்கப்பட்ட தோற்றத்தை உருவாக்க பிக்சலேட்டான வடிவங்களைப் பயன்படுத்துகிறது.
  • Texture: காட்சியின் ஆழத்தை மேம்படுத்துவதற்காக செழுமையான, விவரமான உருப்படிகளை மையமாகக் கொண்டுள்ளது.
