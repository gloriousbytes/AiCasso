Руководство по использованию функции генерации видео

Функция генерации видео в AiCasso позволяет создавать видео, просто описывая, что вы хотите увидеть. Просто напишите свою идею, и AiCasso воплотит её в видео!

Дополнительные опции

Хотите настроить видео? Вот некоторые параметры, с которыми вы можете поиграть:

  • Количество кадров: Выберите, сколько кадров (отдельных изображений) будет в вашем видео. Вы можете выбрать до 25 кадров, по умолчанию — 16.
  • Количество шагов денойзинга: Это контролирует, насколько плавным будет ваше видео. Больше шагов — более плавное видео. Вы можете установить до 50 шагов, по умолчанию — 20.
  • Масштаб руководства: Это помогает видео точно следовать вашему описанию. Чем выше число, тем больше оно следует вашему запросу. Диапазон от 0 до 8, по умолчанию — 8.
  • Пропуск клипов: Пропустите определенные слои для другого стиля видео. По умолчанию слои не пропускаются, но вы можете пропустить до 2.
  • Масштабирование ширины и высоты: Вы можете увеличить видео, изменив разрешение. По умолчанию — 1024x1024 пикселей, но вы можете настроить это под свои нужды.
  • Сила масштабирования: Это изменяет, насколько ваше видео изменяется при увеличении. По умолчанию — 1, но вы можете настроить это, чтобы увидеть больше различий.
  • Масштаб руководства при увеличении: Это направляет процесс увеличения, с диапазоном от 0 до 8, и по умолчанию — 8.
  • Шаги вывода при увеличении: Количество шагов, предпринятых для увеличения видео. Больше шагов делают увеличение более плавным, максимум — 50, по умолчанию — 20.
  • Улучшенная выборка: Хотите более изысканное видео? Включите это. По умолчанию выключено.
  • Семя улучшенной выборки: Используйте это, чтобы генерировать одно и то же видео каждый раз с улучшенной выборкой. Это необязательно и по умолчанию выключено.
  • Кадры в секунду (FPS): Установите скорость вашего видео. По умолчанию — 16 FPS, но вы можете настроить это до 16.
  • Модель видео: Выберите стиль вашего видео. Варианты включают:
    • По умолчанию: Стандартная модель видео.
    • Мультфильм: Мультяшный стиль видео.
    • Естественный: Реалистичный, естественный стиль видео.
    • Анимированный 3D: 3D анимированный стиль видео.

Помните, что эти настройки являются необязательными. Функция генерации видео отлично работает с простым описанием, так что не стесняйтесь экспериментировать и смотреть, что вы можете создать!
