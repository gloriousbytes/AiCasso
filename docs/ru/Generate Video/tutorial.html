Руководство по использованию функции генерации видео

Функция генерации видео в AiCasso позволяет вам создавать видео, просто описывая, что вы хотите увидеть. Просто напишите свою идею, и AiCasso воплотит ее в жизнь в видео!

Дополнительные параметры

Хотите настроить ваше видео? Вот некоторые параметры, с которыми вы можете поэкспериментировать:

  • Количество кадров: Выберите, сколько кадров (отдельных изображений) будет в вашем видео. Вы можете выбрать до 25 кадров, по умолчанию 16.
  • Количество шагов денойзинга: Это контролирует, насколько гладким будет ваше видео. Большее количество шагов означает более гладкие видео. Вы можете установить до 50 шагов, по умолчанию 20.
  • Масштаб управления: Это помогает видео более точно следовать вашему описанию. Чем выше число, тем больше оно соответствует вашему запросу. Диапазон от 0 до 8, по умолчанию 8.
  • Пропуск клипов: Пропустите определенные слои для другого стиля видео. По умолчанию слои не пропускаются, но вы можете пропустить до 2.
  • Ширина и высота увеличения: Вы можете увеличить ваше видео, изменив разрешение. По умолчанию 1024x1024 пикселей, но вы можете настроить это под свои нужды.
  • Сила увеличения: Это изменяет, насколько ваше видео адаптируется при увеличении. По умолчанию 1, но вы можете настроить это, чтобы увидеть больше различий.
  • Масштаб управления увеличением: Это направляет процесс увеличения, диапазон от 0 до 8, по умолчанию 8.
  • Шаги вывода увеличения: Количество шагов, предпринятых для увеличения видео. Большее количество шагов делает увеличение более плавным, максимум 50, по умолчанию 20.
  • Улучшенная выборка: Хотите более изысканное видео? Включите это. По умолчанию выключено.
  • Сид улучшенной выборки: Используйте это, чтобы генерировать одно и то же видео каждый раз с улучшенной выборкой. Это необязательно и по умолчанию выключено.
  • Кадры в секунду (FPS): Установите скорость вашего видео. По умолчанию 16 FPS, но вы можете настроить до 16.
  • Модель видео: Выберите стиль вашего видео. Варианты включают:
    • По умолчанию: Стандартная модель видео.
    • Мультфильм: Мультяшный стиль видео.
    • Естественный: Реалистичный, естественно выглядящий стиль видео.
    • Анимированный 3D: Стиль видео с 3D анимацией.

Помните, эти настройки являются необязательными. Функция генерации видео отлично работает с простым описанием, так что не стесняйтесь экспериментировать и смотреть, что вы можете создать!
