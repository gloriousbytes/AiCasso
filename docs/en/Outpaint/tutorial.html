Tips for Using the Outpaint Feature

The Outpaint feature in AiCasso lets you expand an image by adding new elements around it or extending what’s already there. Just describe what you want, and AiCasso will take care of the rest.

How It Works:

Starting Image:
Describe Your Change:

Just type what you want to add or expand, and AiCasso will apply those changes. You can specify directions like left, right, up, or down, and even details like zooming out or panning.

Example Edits:

Example 1: "Add an iceberg to the right using a translation factor of 0.4."

Result: AiCasso expands the image to the right and adds an iceberg, keeping it consistent with the existing scenery.

Example 2: "Zoom out, keep the consistency of the dark sea at night."

Result: The image zooms out, revealing more of the dark sea, while maintaining the atmosphere of the night scene.

Additional Options

Here are some optional settings you can use to fine-tune your edits:

  • Negative Prompt: If there’s something specific you don’t want in the image, you can tell AiCasso here. For example, you might want to avoid adding certain colors or objects.
  • Walk Type: This decides the direction in which the image expands. You can choose from left, right, up, down, or backward (which zooms out). For example, to pan to the right, you’d select 'right'.
  • Translation Factor: This controls how much the image moves with each step when expanding. A higher value moves the image more, revealing more of the new content. The default is 0.05, but you can adjust it from 0 to 1.
  • Width/Height Translation per Step: These control how far the image moves horizontally (width) or vertically (height) with each step. The default is 32 pixels per step, but you can adjust this based on how much you want the image to shift.
  • Number of Inference or Denoising Steps: This determines how many adjustments AiCasso makes to get the final image. More steps lead to a smoother result, with a default of 20 steps, but you can go up to 50.
  • Number of Interpolation Steps: This controls how smoothly the transition happens between two directions in a single expansion. The default is 60, but you can increase it up to 120 for smoother transitions.

Remember, these settings are optional. AiCasso is designed to work well with just your natural language description. Feel free to experiment and see what works best for your image!
