คู่มือการใช้ฟีเจอร์เปลี่ยนพื้นหลัง

ฟีเจอร์เปลี่ยนพื้นหลังใน AiCasso เหมาะสำหรับเมื่อคุณต้องการเปลี่ยนพื้นหลังในภาพของคุณ ไม่ว่าคุณจะต้องการเพียงแค่ลบพื้นหลังหรือเปลี่ยนเป็นอย่างอื่น AiCasso ทำให้มันง่ายมาก คุณไม่ต้องกังวลเกี่ยวกับเรื่องเทคนิค—เพียงแค่บอกสิ่งที่คุณต้องการ และ AiCasso จะทำที่เหลือให้เอง

ผลลัพธ์ 2:

พารามิเตอร์เสริม

หากคุณต้องการเพิ่มสัมผัสส่วนตัวหรือปรับแต่งผลลัพธ์ นี่คือบางตัวเลือกที่คุณสามารถลองได้:

  • ลบเฉพาะ: หากคุณต้องการเพียงแค่ลบพื้นหลังและไม่เปลี่ยนเป็นอย่างอื่น นี่คือทางเลือกที่เหมาะสม มันเหมือนกับการกดปุ่มเริ่มต้นใหม่
  • Alpha Matting: นี่คือวิธีการที่ซับซ้อนในการพูดว่า "ผสมขอบ" เมื่อคุณเปิดใช้งานมัน จะช่วยทำให้ขอบของวัตถุของคุณดูเรียบเนียนและไม่เหมือนถูกตัดออก โดยค่าเริ่มต้นจะปิดอยู่
  • Threshold ของ Alpha Matting สำหรับ foreground: นี่ควบคุมว่ามีส่วนใดของวัตถุหลัก (เช่น คน) ที่ยังคงมองเห็นได้เมื่อใช้ alpha matting ค่าเริ่มต้นคือ 240 และคุณสามารถปรับให้ขอบนุ่มนวลหรือคมชัดขึ้นได้
  • Threshold ของ Alpha Matting สำหรับ background: นี่ทำงานเหมือนกับ threshold ของ foreground แต่สำหรับพื้นหลัง ค่าเริ่มต้นคือ 20 และมันเกี่ยวกับการทำให้ผสมผสานกับพื้นหลังใหม่ได้อย่างลงตัว
  • ขนาด Erode ของ Alpha Matting: นี่ช่วยลดขอบเล็กน้อยเพื่อทำให้การตัดออกดูสะอาดขึ้น ขนาดเริ่มต้นคือ 5 แต่คุณสามารถปรับให้ได้ลักษณะที่ต้องการ

จำไว้ว่า คุณไม่จำเป็นต้องเครียดเกี่ยวกับการทำให้ทุกอย่างสมบูรณ์แบบ AiCasso ค่อนข้างดีในการเข้าใจสิ่งที่คุณหมายถึง ดังนั้นคุณสามารถพูดได้ตามสบายหรือละเอียดตามที่คุณต้องการ ขอให้สนุกกับการสร้างสรรค์!
