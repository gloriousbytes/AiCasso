Προαιρετικές Παράμετροι

Θέλετε να προσθέσετε λίγο περισσότερη φαντασία ή έλεγχο στα παραγόμενα headshots σας; Ακολουθούν μερικές ωραίες επιλογές που μπορείτε να προσαρμόσετε:

  • Αρνητική Πρόταση: Εδώ μπορείτε να πείτε στο AiCasso τι δεν θέλετε να δείτε στην εικόνα. Για παράδειγμα, αν δεν θέλετε κανένα στοιχείο φόντου, μπορείτε να το αναφέρετε εδώ. Αν το αφήσετε κενό, το AiCasso θα επικεντρωθεί μόνο σε αυτό που θέλετε.
  • Αριθμός Βημάτων Αποθορυβοποίησης: Αυτό ελέγχει πόσο λείο και εκλεπτυσμένο φαίνεται το πρόσωπο. Μπορείτε να επιλέξετε μεταξύ 21, 31 ή 41 βημάτων, με το 41 να είναι η προεπιλογή. Περισσότερα βήματα συνήθως σημαίνουν ένα καθαρότερο, πιο γυαλισμένο αποτέλεσμα.
  • Κλίμακα Καθοδήγησης: Αυτό είναι σαν να δίνετε στο AiCasso μια μικρή επιπλέον ώθηση για να παραμείνει πιστό στην περιγραφή σας. Η κλίμακα κυμαίνεται από 1 έως 20, με το 10 να είναι η προεπιλογή. Υψηλότεροι αριθμοί σημαίνουν ότι το AiCasso θα παραμείνει πιο κοντά σε αυτό που περιγράφετε.
  • Κλίμακα S: Αυτό ρυθμίζει τη δομή του προσώπου. Είναι λίγο τεχνικό, αλλά ουσιαστικά ρυθμίζει πόσο ισχυρά ή μαλακά είναι τα χαρακτηριστικά του προσώπου. Η προεπιλογή είναι 1.0, και μπορείτε να πάτε οπουδήποτε από 0.0 έως 2.0 ανάλογα με την εμφάνιση που επιθυμείτε.

Θυμηθείτε, αυτές οι παράμετροι είναι εντελώς προαιρετικές. Το AiCasso είναι αρκετά συγχωρητικό, οπότε δεν χρειάζεται να είστε πολύ ακριβείς ή γραμματικά σωστοί. Απλά περιγράψτε αυτό που θέλετε με δικά σας λόγια, και το AiCasso θα κάνει τη μαγεία του. Καλή διασκέδαση δημιουργώντας τα headshots σας!
