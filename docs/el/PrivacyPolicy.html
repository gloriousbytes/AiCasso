Πολιτική Απορρήτου για το AiCasso

Ημερομηνία Ισχύος: 13 Σεπτεμβρίου 2024

Στο AiCasso, η ιδιωτικότητά σας είναι προτεραιότητά μας. Αυτή η Πολιτική Απορρήτου περιγράφει πώς χειριζόμαστε τις πληροφορίες που παρέχετε κατά τη χρήση της εφαρμογής μας.

  1. Πληροφορίες που Συλλέγουμε
    • Εικόνες Εισόδου: Συλλέγουμε τις εικόνες που ανεβάζετε στο AiCasso αποκλειστικά για να παρέχουμε τις υπηρεσίες μας.
  2. Πώς Χρησιμοποιούμε τις Πληροφορίες σας
    • Παροχή Υπηρεσιών: Οι εικόνες εισόδου χρησιμοποιούνται αποκλειστικά για την εκτέλεση των λειτουργιών που προσφέρει το AiCasso.
  3. Διατήρηση Δεδομένων
    • Άμεση Διαγραφή: Όλες οι εικόνες εισόδου διαγράφονται άμεσα μετά την επεξεργασία τους και την εκπλήρωση του σκοπού τους.
  4. Κοινοποίηση Δεδομένων
    • Καμία Κοινοποίηση σε Τρίτους: Δεν μοιραζόμαστε, δεν πωλούμε και δεν αποκαλύπτουμε τις εικόνες σας ή οποιαδήποτε προσωπικά δεδομένα σε τρίτους.
  5. Μέτρα Ασφαλείας
    • Προστασία Δεδομένων: Χρησιμοποιούμε μέτρα ασφαλείας που πληρούν τα πρότυπα της βιομηχανίας για να προστατεύσουμε τις εικόνες σας κατά την επεξεργασία.
  6. Αλλαγές σε Αυτή την Πολιτική Απορρήτου
    • Ενημερώσεις Πολιτικής: Μπορεί να ενημερώνουμε αυτή την Πολιτική Απορρήτου από καιρό σε καιρό. Οποιεσδήποτε αλλαγές θα αποτυπώνονται με μια ενημερωμένη ημερομηνία ισχύος.
  7. Επικοινωνήστε Μαζί μας

    Εάν έχετε οποιαδήποτε ερώτηση ή ανησυχία σχετικά με αυτή την Πολιτική Απορρήτου, παρακαλούμε επικοινωνήστε μαζί μας στο:
