Πολιτική Απορρήτου για το AiCasso

Ημερομηνία Ισχύος: 13 Σεπτεμβρίου 2024

Στο AiCasso, η ιδιωτικότητά σας είναι προτεραιότητά μας. Αυτή η Πολιτική Απορρήτου περιγράφει πώς χειριζόμαστε τις πληροφορίες που παρέχετε κατά τη χρήση της εφαρμογής μας.

  1. Πληροφορίες που Συλλέγουμε
    • Εικόνες Εισόδου: Συλλέγουμε τις εικόνες που ανεβάζετε στο AiCasso αποκλειστικά για να παρέχουμε τις υπηρεσίες μας.
    • Δεδομένα Προσώπου: Εάν ανεβάσετε μια εικόνα που περιέχει πρόσωπα, επεξεργαζόμαστε τα χαρακτηριστικά του προσώπου για να δημιουργήσουμε προσαρμοσμένες εικόνες με βάση τα χαρακτηριστικά που επιλέγει ο χρήστης. Αυτά τα δεδομένα προσώπου εξάγονται από τις εικόνες που ανεβάζετε και χρησιμοποιούνται αποκλειστικά για τον σκοπό της δημιουργίας εικόνας.
  2. Πώς Χρησιμοποιούμε τις Πληροφορίες σας
    • Παροχή Υπηρεσιών: Οι εικόνες εισόδου, συμπεριλαμβανομένων των δεδομένων προσώπου, χρησιμοποιούνται αποκλειστικά για την εκτέλεση των λειτουργιών που προσφέρει το AiCasso. Αυτό περιλαμβάνει τη δημιουργία νέων εικόνων με βάση τα χαρακτηριστικά που επιλέγετε.
  3. Διατήρηση Δεδομένων
    • Άμεση Διαγραφή Ανεβασμένων Εικόνων: Όλες οι εικόνες εισόδου διαγράφονται άμεσα μετά την επεξεργασία τους και την εκπλήρωση του σκοπού τους.
    • Προσωρινή Αποθήκευση Δημιουργημένων Εικόνων: Οι δημιουργημένες εικόνες αποθηκεύονται στην υπηρεσία cloud μας και είναι διαθέσιμες στον χρήστη για 24 ώρες. Αυτές οι εικόνες διαγράφονται αυτόματα μετά από αυτή την περίοδο.
    • Διαχείριση Αποθήκευσης Cloud: Η αποθήκευση cloud μας καθαρίζεται κάθε 24 ώρες για να διασφαλιστεί ότι δεν παραμένουν υπολείμματα δεδομένων.
  4. Κοινοποίηση Δεδομένων
    • Καμία Κοινοποίηση σε Τρίτους: Δεν κοινοποιούμε, δεν πωλούμε και δεν αποκαλύπτουμε τις εικόνες σας, τα δεδομένα προσώπου ή οποιαδήποτε προσωπικά δεδομένα σε τρίτους. Όλη η επεξεργασία δεδομένων γίνεται εσωτερικά στο AiCasso.
  5. Μέτρα Ασφαλείας
    • Προστασία Δεδομένων: Χρησιμοποιούμε μέτρα ασφαλείας που πληρούν τα πρότυπα της βιομηχανίας για να προστατεύσουμε τις εικόνες και τα δεδομένα προσώπου σας κατά την επεξεργασία και ενώ είναι αποθηκευμένα στις υπηρεσίες cloud μας.
  6. Αλλαγές σε Αυτή την Πολιτική Απορρήτου
    • Ενημερώσεις Πολιτικής: Μπορεί να ενημερώνουμε αυτή την Πολιτική Απορρήτου από καιρό σε καιρό. Οποιεσδήποτε αλλαγές θα αντικατοπτρίζονται με μια ενημερωμένη ημερομηνία ισχύος.
