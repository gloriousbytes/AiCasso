Οδηγός Χρήσης του Χαρακτηριστικού Μετατροπής Εικόνας σε Βίντεο

Το χαρακτηριστικό Μετατροπής Εικόνας σε Βίντεο στο AiCasso σας επιτρέπει να μετατρέψετε μια μόνο εικόνα σε ένα δυναμικό βίντεο. Απλώς περιγράψτε τι θέλετε να δείτε στο βίντεό σας και το AiCasso θα το πραγματοποιήσει!

Πώς Λειτουργεί:

Είσοδος Εικόνας:
Πρόταση 1:

"Ένα χαριτωμένο κουτάβι στο πάρκο"

Αποτέλεσμα 1:
Πρόταση 2:

"Ένα χαριτωμένο κουτάβι στο πάρκο, χρησιμοποιήστε μέγιστο FPS, μέγιστους καρέ και μέγιστη κίνηση"

Αποτέλεσμα 2:

Προαιρετικές Παράμετροι

Αν θέλετε να γίνετε πιο συγκεκριμένοι με το βίντεό σας, εδώ είναι μερικές ρυθμίσεις που μπορείτε να προσαρμόσετε:

  • Αρνητική Πρόταση: Περιγράψτε τι *δεν* θέλετε στο βίντεο. Αν μείνει κενό, το AiCasso θα επικεντρωθεί μόνο στην θετική πρόταση.
  • Αριθμός Καρέ: Επιλέξτε πόσα καρέ (ατομικές εικόνες) θα έχει το βίντεό σας. Μπορείτε να επιλέξετε έως 25 καρέ, με 16 να είναι η προεπιλογή.
  • Βήματα Αποθορυβοποίησης: Αυτό ελέγχει πόσο ομαλό φαίνεται το βίντεό σας. Περισσότερα βήματα σημαίνουν πιο καθαρό βίντεο. Μπορείτε να ορίσετε έως 50 βήματα, με 20 να είναι η προεπιλογή.
  • Κλίμακα Καθοδήγησης: Αυτό βοηθά το βίντεο να παραμείνει κοντά στην περιγραφή σας. Η κλίμακα είναι από 1.0 έως 8.0, και μπορείτε να προσαρμόσετε τις αρχικές και τελικές τιμές για να καθοδηγήσετε την εξέλιξη του βίντεο.
  • Κίνηση: Ρυθμίστε πόση κίνηση υπάρχει στο βίντεό σας. Μια υψηλότερη τιμή σημαίνει περισσότερη δράση. Η κλίμακα είναι από 0 έως 127, με 60 να είναι η προεπιλογή.
  • Δύναμη Αυξησης Θορύβου: Προσθέστε λίγο θόρυβο στο βίντεό σας για να του δώσετε μια πιο υφή. Η κλίμακα είναι από 0 έως 1, με 0 να είναι η προεπιλογή.
  • Καρέ ανά Δευτερόλεπτο (FPS): Ορίστε πόσο γρήγορα παίζει το βίντεό σας. Η προεπιλογή είναι 16 FPS, αλλά μπορείτε να πάτε έως 16 για ταχύτερη αναπαραγωγή.
  • Μέθοδος Ενδιάμεσης Εξίσωσης: Επιλέξτε πώς αναμειγνύονται τα καρέ. Η προεπιλογή είναι "lerp" (γραμμική ενδιάμεση εξίσωση), αλλά μπορείτε επίσης να χρησιμοποιήσετε "slerp" (σφαιρική γραμμική ενδιάμεση εξίσωση) για διαφορετικά εφέ.
  • Παράλειψη Κλιπ: Παράλειψη ορισμένων επιπέδων για διαφορετικό στυλ βίντεο. Μπορείτε να παραλείψετε έως 2 επίπεδα ή να το αφήσετε κενό για την προεπιλεγμένη συμπεριφορά.
