Οδηγός Χρήσης της Λειτουργίας Δημιουργίας Βίντεο

Η λειτουργία Δημιουργίας Βίντεο στο AiCasso σας επιτρέπει να δημιουργείτε βίντεο απλά περιγράφοντας αυτό που θέλετε να δείτε. Απλά πληκτρολογήστε την ιδέα σας και το AiCasso θα την ζωντανέψει σε ένα βίντεο!

Πώς Λειτουργεί:

Ερώτημα:

"Ένας 3D κινούμενος κόκορας σε ένα χωράφι μόνος του, τρώγοντας και ευτυχισμένος. Κάντε το ένα πορτραίτο βίντεο με 50 βήματα αποθορυβοποίησης."

Αποτέλεσμα:
Ερώτημα:

"Μια καρτούν εικόνα ενός κόκορα σε ένα χωράφι σε τοπίο διαστάσεων."

Αποτέλεσμα:
Ερώτημα:

"Ένας κόκορας σε ένα χωράφι."

Αποτέλεσμα:
Ερώτημα:

"Ένας κόκορας που είναι στο χωράφι μόνος του. Δώστε του μια φυσική αίσθηση."

Αποτέλεσμα:

Επιπλέον Επιλογές

Θέλετε να ρυθμίσετε το βίντεό σας; Εδώ είναι μερικές ρυθμίσεις που μπορείτε να πειραματιστείτε:

  • Αριθμός Καρέ: Επιλέξτε πόσα καρέ (μεμονωμένες εικόνες) θα έχει το βίντεό σας. Μπορείτε να φτάσετε έως και 25 καρέ, με 16 ως προεπιλογή.
  • Αριθμός Βημάτων Αποθορυβοποίησης: Αυτό ελέγχει πόσο ομαλό φαίνεται το βίντεό σας. Περισσότερα βήματα σημαίνουν πιο ομαλά βίντεο. Μπορείτε να ορίσετε έως και 50 βήματα, με 20 ως προεπιλογή.
  • Κλίμακα Καθοδήγησης: Αυτό βοηθά το βίντεο να παραμένει κοντά στην περιγραφή σας. Όσο υψηλότερος είναι ο αριθμός, τόσο περισσότερο ακολουθεί την ερώτησή σας. Η κλίμακα είναι από 0 έως 8, με 8 να είναι η προεπιλογή.
  • Παράλειψη Κλιπ: Παράλειψη ορισμένων επιπέδων για διαφορετικό στυλ βίντεο. Από προεπιλογή, δεν παραλείπονται επίπεδα, αλλά μπορείτε να παραλείψετε έως και 2.
  • Διαστάσεις Μεγέθυνσης: Μπορείτε να κάνετε το βίντεό σας μεγαλύτερο ρυθμίζοντας την ανάλυση. Η προεπιλογή είναι 1024x1024 pixels, αλλά μπορείτε να το προσαρμόσετε στις ανάγκες σας.
  • Δύναμη Μεγέθυνσης: Αυτό αλλάζει πόσο προσαρμόζεται το βίντεό σας όταν μεγεθύνεται. Η προεπιλογή είναι 1, αλλά μπορείτε να το ρυθμίσετε για να δείτε περισσότερες διαφορές.
  • Κλίμακα Καθοδήγησης Μεγέθυνσης: Αυτό καθοδηγεί τη διαδικασία μεγέθυνσης, με κλίμακα από 0 έως 8, και προεπιλογή 8.
  • Βήματα Επαναληπτικής Μεγέθυνσης: Ο αριθμός των βημάτων που γίνονται για να μεγεθυνθεί το βίντεο. Περισσότερα βήματα κάνουν τη μεγέθυνση πιο ομαλή, με μέγιστο 50 και προεπιλογή 20.
  • Βελτιωμένο Δείγμα: Θέλετε ένα πιο εκλεπτυσμένο βίντεο; Ενεργοποιήστε το. Είναι απενεργοποιημένο από προεπιλογή.
  • Σπόρος Βελτιωμένου Δείγματος: Χρησιμοποιήστε το για να δημιουργήσετε το ίδιο βίντεο κάθε φορά με βελτιωμένο δείγμα. Είναι προαιρετικό και απενεργοποιημένο από προεπιλογή.
  • Καρέ ανά Δευτερόλεπτο (FPS): Ρυθμίστε την ταχύτητα του βίντεό σας. Η προεπιλογή είναι 16 FPS, αλλά μπορείτε να την προσαρμόσετε έως και 16.
  • Μοντέλο Βίντεο: Επιλέξτε το στυλ του βίντεό σας. Οι επιλογές περιλαμβάνουν:
    • Προεπιλογή: Ένα τυπικό μοντέλο βίντεο.
    • Καρτούν: Ένα καρτουνίστικο στυλ βίντεο.
    • Φυσικό: Ένα ρεαλιστικό, φυσικό στυλ βίντεο.
    • Κινούμενο 3D: Ένα στυλ βίντεο 3D κινούμενης εικόνας.

Θυμηθείτε, αυτές οι ρυθμίσεις είναι προαιρετικές. Η λειτουργία Δημιουργίας Βίντεο λειτουργεί εξαιρετικά με απλή περιγραφή, οπότε μη διστάσετε να πειραματιστείτε και να δείτε τι μπορείτε να δημιουργήσετε!
