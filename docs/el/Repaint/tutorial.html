Συμβουλές για τη Λειτουργία Επαναχρωματισμού

Η χρήση της λειτουργίας Inpaint στο AiCasso σας επιτρέπει να τροποποιείτε συγκεκριμένες περιοχές μιας εικόνας γεμίζοντας ή αντικαθιστώντας τις. Είτε θέλετε να βελτιώσετε μια υπάρχουσα εικόνα είτε να αλλάξετε εντελώς ορισμένα τμήματα, το AiCasso καθιστά τη διαδικασία απλή. Απλώς επιλέξτε τις περιοχές που θέλετε να αλλάξετε, δώστε μια προτροπή και αφήστε το AiCasso να κάνει τη μαγεία του.

Επιλογές:

Δύναμη:

Η επιλογή δύναμης ελέγχει πόσο από την αρχική εικόνα αλλάζει από την προτροπή. Κυμαίνεται από 0 έως 1. Μια τιμή 0 σημαίνει ότι η αρχική εικόνα αλλάζει ελάχιστα, ενώ μια τιμή 1 σημαίνει ότι η αρχική εικόνα αντικαθίσταται σχεδόν πλήρως από τις νέες λεπτομέρειες που περιγράφετε. Σκεφτείτε το ως το πόσο 'ισχυρές' θα είναι οι αλλαγές σας.

Βελτίωση Στυλ:

Η επιλογή βελτίωσης στυλ σας επιτρέπει να προσθέσετε ένα συγκεκριμένο καλλιτεχνικό ή θεματικό στυλ στις τροποποιημένες περιοχές. Μπορείτε να επιλέξετε από μια ποικιλία στυλ όπως:

Πώς Λειτουργεί:

Εικόνα Εισόδου:
Ζωγραφίστε τις Περιοχές που Πρέπει να Γεμιστούν ή να Αντικατασταθούν:

Παραδείγματα:

Προτροπή 1: "Μια υπερρεαλιστική εικόνα μιας γάτας στο πάρκο."

Αποτέλεσμα: Το AiCasso θα δημιουργήσει μια υπερρεαλιστική γάτα στο πάρκο, γεμίζοντας τις ζωγραφισμένες περιοχές ομαλά με ρεαλιστικές υφές και φωτισμό.

Προτροπή 2: "Μια γάτα στο πάρκο με υπερρεαλιστικό στυλ και δύναμη 0.7."

Αποτέλεσμα: Αυτό θα δημιουργήσει μια υπερρεαλιστική γάτα στο πάρκο, με τη δύναμη τροποποίησης ρυθμισμένη στο 0.7, συνδυάζοντας τα νέα στοιχεία ομαλά με την αρχική εικόνα.
