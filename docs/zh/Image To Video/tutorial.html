可选参数

如果您想让视频更具体，这里有一些可以调整的设置：

  • 负面提示：描述您*不*希望在视频中出现的内容。如果留空，AiCasso 将只关注正面提示。
  • 帧数：选择视频将包含多少帧（单独的图像）。您可以选择最多 25 帧，默认值为 16 帧。
  • 去噪步骤：这控制视频的平滑程度。更多的步骤意味着更清晰的视频。您可以设置最多 50 步，默认值为 20 步。
  • 引导比例：这有助于视频紧密贴合您的描述。范围从 1.0 到 8.0，您可以调整起始和结束值以指导视频的演变。
  • 运动：调整视频中的运动量。更高的值意味着更多的动作。范围从 0 到 127，默认值为 60。
  • 噪声增强强度：为视频添加一些噪声，以使其具有更丰富的质感。范围从 0 到 1，默认值为 0。
  • 每秒帧数（FPS）：设置视频播放的速度。默认值为 16 FPS，但您可以选择更高的值以获得更快的播放。
  • 插值方法：选择帧之间的混合方式。默认值为 "lerp"（线性插值），但您也可以使用 "slerp"（球面线性插值）以获得不同的效果。
  • 跳过剪辑：跳过某些层以获得不同的视频风格。您可以跳过最多 2 层，或将其留空以使用默认行为。

记住，AiCasso 非常聪明！您不必将每个细节都做到完美或使用正确的语法。只需描述您想要的内容，AiCasso 将完成其余工作。祝您创建视频愉快！
