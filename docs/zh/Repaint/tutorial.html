在 AiCasso 中使用重绘功能可以通过填充或替换来修改图像的特定区域。无论您是想增强现有图像还是完全改变某些部分，AiCasso 都能让过程变得简单。只需选择您想要更改的区域，提供提示，然后让 AiCasso 展现其魔力。

选项：

强度：

强度选项控制提示对原始图像的更改程度。范围从 0 到 1。值为 0 表示原始图像几乎未被改变，而值为 1 表示原始图像几乎完全被您描述的新细节替换。可以将其视为更改的“强度”。
