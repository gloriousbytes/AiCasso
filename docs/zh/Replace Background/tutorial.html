可选参数

如果您想添加个人风格或微调结果，这里有一些选项供您选择：

  • 仅移除：如果您只想移除背景而不替换任何东西，这是您的首选选项。就像按下一个清除按钮。
  • Alpha 抠图：这是一种让边缘更平滑的高级方式。当您开启它时，它有助于使主体的边缘看起来更平滑而不那么生硬。默认情况下，它是关闭的。
  • Alpha 抠图前景阈值：这控制了使用 alpha 抠图时主体（如人物）保留多少可见部分。默认值是 240，您可以调整它以使边缘更柔和或更锐利。
  • Alpha 抠图背景阈值：这与前景阈值的作用相同，但针对背景。默认值是 20，旨在与新背景完美融合。
  • Alpha 抠图侵蚀大小：这有助于稍微缩小边缘，使剪切效果更干净。默认大小是 5，但您可以调整它以获得所需的确切效果。

记住，您不需要为达到完美而感到压力。AiCasso 非常擅长理解您的意思，所以您可以随意或详细地描述。祝您创作愉快！
