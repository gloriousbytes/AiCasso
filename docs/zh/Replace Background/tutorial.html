可选参数

如果您想添加个人风格或微调结果，这里有一些您可以尝试的选项：

  • 仅删除：如果您只想删除背景而不替换任何东西，这是您的首选选项。这就像按下一个干净的重置按钮。
  • Alpha 合成：这是一个花哨的说法，意思是“平滑边缘”。当您开启此功能时，它有助于使主题的边缘看起来更平滑，而不是生硬的切割。默认情况下，它是关闭的。
  • Alpha 合成前景阈值：这控制在使用 alpha 合成时，主要主题（如人）保持可见的程度。默认值为 240，您可以调整它以使边缘更柔和或更锐利。
  • Alpha 合成背景阈值：这与前景阈值的作用相同，但适用于背景。默认值为 20，旨在与新背景实现完美融合。
  • Alpha 合成腐蚀大小：这有助于稍微缩小边缘，使切割看起来更干净。默认大小为 5，但您可以调整它以获得所需的确切效果。

请记住，您不需要担心一切都完美。AiCasso在理解您的意思方面相当不错，因此请随意保持随意或详细。祝您创作愉快！
