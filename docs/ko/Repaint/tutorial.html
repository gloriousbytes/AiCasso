재페인트 기능 팁

AiCasso의 인페인트 기능을 사용하면 이미지의 특정 영역을 채우거나 교체하여 수정할 수 있습니다. 기존 이미지를 향상시키거나 특정 섹션을 완전히 변경하고 싶든, AiCasso는 과정을 간단하게 만들어 줍니다. 변경하고 싶은 영역을 선택하고, 프롬프트를 제공한 후 AiCasso가 마법을 부리도록 하세요.

옵션:

강도:

강도 옵션은 프롬프트에 의해 원본 이미지가 얼마나 변경되는지를 제어합니다. 0에서 1까지의 범위를 가집니다. 0의 값은 원본 이미지가 거의 변경되지 않음을 의미하고, 1의 값은 원본 이미지가 설명한 새로운 세부 사항으로 거의 완전히 교체됨을 의미합니다. 변경의 '강도'를 생각해 보세요.

스타일 향상:

스타일 향상 옵션은 수정된 영역에 특정 예술적 또는 주제적 스타일을 추가할 수 있게 해줍니다. 다음과 같은 다양한 스타일 중에서 선택할 수 있습니다:

작동 방식:

입력 이미지:
채우거나 교체할 영역 그리기:

예시:

프롬프트 1: "공원에서 고양이의 하이퍼 리얼리즘 이미지."

결과: AiCasso는 공원에서 하이퍼 리얼리즘 고양이를 생성하며, 그려진 영역을 사실적인 질감과 조명으로 매끄럽게 채웁니다.

프롬프트 2: "하이퍼 리얼리즘 스타일의 공원에서 고양이, 강도 0.7."

결과: 이는 강도 0.7로 설정된 하이퍼 리얼리즘 고양이를 생성하며, 새로운 요소가 원본 이미지에 매끄럽게 혼합됩니다.

스타일 향상

어떤 스타일을 선택해야 할지 모르겠나요? 걱정하지 마세요! 문법적으로 맞지 않더라도 떠오르는 스타일을 언급하면 AiCasso가 사용 가능한 스타일 목록에서 알아낼 것입니다.
