AiCasso의 개인정보 처리방침

발효일: 2024년 9월 13일

AiCasso에서는 귀하의 개인정보를 최우선으로 생각합니다. 이 개인정보 처리방침은 귀하가 우리 애플리케이션을 사용할 때 제공하는 정보를 어떻게 처리하는지를 설명합니다.

  1. 수집하는 정보
    • 입력 이미지: 우리는 AiCasso에 업로드한 이미지를 서비스 제공을 위해 수집합니다.
    • 얼굴 데이터: 얼굴이 포함된 이미지를 업로드하는 경우, 우리는 사용자 선택 기능에 따라 맞춤형 이미지를 생성하기 위해 얼굴 특징을 처리합니다. 이 얼굴 데이터는 귀하가 업로드한 이미지에서 추출되며 이미지 생성 목적에만 사용됩니다.
  2. 정보 사용 방법
    • 서비스 제공: 입력 이미지와 얼굴 데이터는 AiCasso에서 제공하는 기능을 수행하는 데만 사용됩니다. 여기에는 사용자가 선택한 기능에 따라 새로운 이미지를 생성하는 것이 포함됩니다.
  3. 데이터 보존
    • 업로드된 이미지의 즉시 삭제: 모든 입력 이미지는 처리 후 즉시 삭제되며, 그 목적이 달성됩니다.
    • 생성된 이미지의 임시 저장: 생성된 이미지는 우리 클라우드 서비스에 저장되며 사용자에게 24시간 동안 제공됩니다. 이 기간이 지나면 자동으로 삭제됩니다.
    • 클라우드 저장소 관리: 우리의 클라우드 저장소는 24시간마다 정리되어 잔여 데이터가 남지 않도록 합니다.
  4. 데이터 공유
    • 제3자 공유 없음: 우리는 귀하의 이미지, 얼굴 데이터 또는 개인 데이터를 제3자와 공유, 판매 또는 공개하지 않습니다. 모든 데이터 처리는 AiCasso 내부에서 처리됩니다.
  5. 보안 조치
    • 데이터 보호: 우리는 처리 중 및 클라우드 서비스에 저장되는 동안 귀하의 이미지와 얼굴 데이터를 보호하기 위해 업계 표준 보안 조치를 취합니다.
  6. 이 개인정보 처리방침의 변경
    • 정책 업데이트: 우리는 이 개인정보 처리방침을 수시로 업데이트할 수 있습니다. 변경 사항은 업데이트된 발효일로 반영됩니다.
