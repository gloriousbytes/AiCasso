AiCasso 개인정보 처리방침

시행일: 2024년 9월 13일

AiCasso에서는 귀하의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보 처리방침은 당사 애플리케이션 사용 시 제공하신 정보를 어떻게 처리하는지 설명합니다.

  1. 수집하는 정보
    • 입력 이미지: AiCasso 서비스를 제공하기 위해 귀하가 업로드한 이미지를 수집합니다.
  2. 정보 사용 방법
    • 서비스 제공: 입력 이미지는 AiCasso가 제공하는 기능을 수행하기 위해서만 사용됩니다.
  3. 데이터 보유
    • 즉시 삭제: 모든 입력 이미지는 처리 후 즉시 삭제되며, 의도된 목적이 달성됩니다.
  4. 데이터 공유
    • 제3자 공유 없음: 귀하의 이미지나 개인 데이터를 제3자에게 공유, 판매 또는 공개하지 않습니다.
  5. 보안 조치
    • 데이터 보호: 처리 중 귀하의 이미지를 보호하기 위해 산업 표준 보안 조치를 사용합니다.
  6. 개인정보 처리방침 변경
    • 정책 업데이트: 본 개인정보 처리방침은 수시로 업데이트될 수 있으며, 변경 사항은 업데이트된 시행일과 함께 반영됩니다.
  7. 문의하기

    본 개인정보 처리방침에 대한 질문이나 우려 사항이 있으시면 다음으로 연락해 주십시오:
